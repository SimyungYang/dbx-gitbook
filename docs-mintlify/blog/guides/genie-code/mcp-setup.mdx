## MCP 개요

### MCP란 무엇인가?

MCP(Model Context Protocol)는 **Anthropic이 개발한 오픈소스 프로토콜** 로, AI 에이전트가 외부 도구, 데이터 소스, 워크플로에 접근하기 위한 **표준 인터페이스** 입니다.

### MCP가 Genie Code를 어떻게 변환하는가

MCP를 연결하면 Genie Code는 **워크스페이스의 경계를 넘어서는 AI 에이전트** 로 변환됩니다:

| MCP 없는 Genie Code | MCP가 있는 Genie Code |
|---------------------|----------------------|
| 분석 완료 → 사용자가 Slack을 열어 결과를 복사/붙여넣기 | "Slack #team에 결과 공유해줘" 한 마디로 전송 |
| 이슈 발견 → 사용자가 JIRA를 열어 티켓 수동 생성 | "JIRA에 Bug 티켓 만들어줘" 로 자동 생성 |

> **참고**
MCP 연동의 진정한 가치는 여러 도구를 **하나의 워크플로로 연결** 하는 데 있습니다.


### 핵심 아키텍처

| 참여자 | 역할 | Databricks 예시 |
|--------|------|-----------------|
| **MCP Host** | AI 애플리케이션 | Genie Code, AI Playground |
| **MCP Client** | 연결 유지, 컨텍스트 획득 | Genie Code 내부 클라이언트 |
| **MCP Server** | 도구, 리소스, 프롬프트 제공 | GitHub MCP, UC Functions 등 |

### MCP 서버가 제공하는 3가지 기본 구성 요소

| 구성 요소 | 설명 |
|-----------|------|
| **Tools** | AI가 호출할 수 있는 실행 가능한 함수 |
| **Resources** | AI에 컨텍스트를 제공하는 데이터 소스 |
| **Prompts** | 재사용 가능한 상호작용 템플릿 |

## Databricks에서 MCP 서버 설정

### 유형 1: Managed MCP (관리형)

Databricks가 사전 구성한 즉시 사용 가능한 MCP 서버입니다.

| 서버 | 용도 |
|------|------|
| **Unity Catalog Functions** | 사전 정의된 SQL 함수 실행 |
| **Vector Search** | 벡터 검색 인덱스 쿼리 |
| **Genie Space** | 자연어 데이터 분석 (읽기 전용) |
| **Databricks SQL** | AI 생성 SQL 실행 (읽기/쓰기) |

### 유형 2: External MCP (외부)

Unity Catalog Connection을 통해 외부 MCP 서버에 안전하게 연결합니다.

**지원되는 연결 방법**: Managed OAuth (권장), Databricks Marketplace, Custom HTTP Connection, Dynamic Client Registration (실험적)

### 유형 3: Custom MCP (커스텀)

자체 MCP 서버를 **Databricks App** 으로 호스팅합니다.

> **주의**
커스텀 MCP 앱은 **stateless 아키텍처** 로 구현해야 하며, 동일 워크스페이스 내에 배포해야 합니다.

## Genie Code에서 MCP 활용

### MCP 서버를 Genie Code에 연결하기

> **주의**
MCP 서버는 **Genie Code Agent 모드에서만** 지원됩니다.


**설정 단계:**

1. Genie Code 패널 > **Settings** > **MCP Servers**
2. **Add Server** 선택
3. 서버 유형 선택 (UC Functions, Vector Search, Genie Spaces, UC Connections, Databricks Apps)
4. **Save**

### .mcp_servers.json 파일

Workspace의 `.assistant/.mcp_servers.json` 파일에 구성이 저장됩니다.

```json
[
  {"type": "uc_function", "uc_function_name": "catalog.schema.function_name"},
  {"type": "vector_search", "vector_search_index": "catalog.schema.index_name"},
  {"type": "genie_space", "genie_space_id": "01ef1234567890abcdef"},
  {"type": "uc_connection", "connection_name": "github-mcp-connection"},
  {"type": "databricks_app", "app_name": "my-custom-mcp-app"}
]
```

| 유형 | `type` 값 | 필수 필드 |
|------|----------|---------|
| **Unity Catalog Function** | `uc_function` | `uc_function_name` |
| **Vector Search Index** | `vector_search` | `vector_search_index` |
| **Genie Space** | `genie_space` | `genie_space_id` |
| **UC Connection (외부 MCP)** | `uc_connection` | `connection_name` |
| **Databricks App** | `databricks_app` | `app_name` |

> **주의**
**최대 20개 Tool** 제한을 초과하지 않도록 필요한 서버만 등록하세요.
