## 💻 샘플 코드

```sql
-- Foreign Catalog (Snowflake/MySQL)
CREATE <OBJECT_TYPE> IF NOT EXISTS workshop_<name>.<schema>.<object>
  (<column_spec>)
  COMMENT '<설명>'
  TBLPROPERTIES ('owner' = current_user());

-- 메타데이터 확인
DESCRIBE EXTENDED workshop_<name>.<schema>.<object>;
```
