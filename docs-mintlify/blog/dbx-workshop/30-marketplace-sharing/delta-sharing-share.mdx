## 💻 샘플 코드

```sql
-- Share 만들기
CREATE SHARE workshop_share;
ALTER SHARE workshop_share ADD TABLE workshop_<name>.default.events;

-- Recipient 생성 (D2D 또는 Open)
CREATE RECIPIENT partner_<name>
  USING ID '<partner-sharing-identifier>';  -- D2D
-- 또는 Open Sharing: CREATE RECIPIENT ... (no USING ID)

-- Recipient에 Share 권한 부여
GRANT SELECT ON SHARE workshop_share TO RECIPIENT partner_<name>;
```
