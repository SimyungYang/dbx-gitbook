| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```yaml
# Jobs UI에서 만든 Job을 YAML로 보면 다음 형태
name: workshop-etl
tasks:
  - task_key: ingest
    notebook_task:
      notebook_path: /Workspace/Users/<email>/notebooks/ingest
    job_cluster_key: main
job_clusters:
  - job_cluster_key: main
    new_cluster:
      spark_version: 15.4.x-scala2.12
      node_type_id: i3.xlarge
      num_workers: 2
schedule:
  quartz_cron_expression: '0 0 4 * * ?'
  timezone_id: Asia/Seoul
```
