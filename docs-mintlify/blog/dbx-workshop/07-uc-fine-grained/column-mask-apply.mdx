## 💻 샘플 코드

```sql
-- Row Filter Function
CREATE FUNCTION workshop_<name>.default.region_filter(region STRING)
RETURNS BOOLEAN
RETURN region = current_user_region();

-- 테이블에 Row Filter 적용
ALTER TABLE workshop_<name>.default.sales
  SET ROW FILTER workshop_<name>.default.region_filter
  ON (region);
```
