## 💻 샘플 코드

```sql
-- Agent (Endpoint) 권한
GRANT EXECUTE ON ENDPOINT <agent-endpoint> TO `users`;

-- FMAPI / Foundation Model Endpoint 권한
GRANT USAGE ON SERVING ENDPOINT databricks-meta-llama-3-1-70b-instruct TO `users`;
```
