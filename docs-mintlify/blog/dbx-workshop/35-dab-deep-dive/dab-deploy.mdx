## 🧭 이 기능은 무엇이고 언제 쓰나

본 Lab은 **bundle deploy** 을 단일 기능 단위로 직접 수행하며 동작을 체감하는 것이 목적입니다. `M35. Databricks Asset Bundle (DAB) — Deep Dive` 의 핵심 흐름 안에서 이 단계가 어떤 역할을 하는지 위치를 잡습니다.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```yaml
# databricks.yml — 최상위
bundle:
  name: workshop_bundle

include:
  - resources/jobs/*.yml
  - resources/pipelines/*.yml

targets:
  dev:
    workspace:
      profile: workshop-dev
    default: true
  prod:
    workspace:
      profile: workshop-prod
```
