## 💻 샘플 코드

```python
from databricks.vector_search.client import VectorSearchClient

vsc = VectorSearchClient()

# Endpoint 생성
vsc.create_endpoint(name='workshop-vs', endpoint_type='STANDARD')

# Delta Sync Index
vsc.create_delta_sync_index(
    endpoint_name='workshop-vs',
    index_name='workshop_<name>.default.docs_idx',
    source_table_name='workshop_<name>.default.docs',
    pipeline_type='TRIGGERED',
    primary_key='doc_id',
    embedding_source_column='content',
    embedding_model_endpoint_name='databricks-bge-large-en',
)
```
