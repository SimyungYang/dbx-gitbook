## 🧭 이 기능은 무엇이고 언제 쓰나

Identity Federation은 Account-level User/Group을 Workspace에 자동 할당하는 기능입니다. 비활성화면 Workspace마다 수동 할당이 필요합니다.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Identity Federation 활성 (권장)
  → User/Group을 Account Console에서 만들면 자동으로 모든 Workspace에 노출됨
  → Workspace Admin은 "할당" 만 (생성 X)

Identity Federation 비활성 (legacy)
  → User/Group을 각 Workspace에서 별도 생성
  → 동일 User가 Workspace마다 다른 ID
```
