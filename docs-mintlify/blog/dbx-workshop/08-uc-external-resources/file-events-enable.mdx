## 💻 샘플 코드

```sql
-- Storage Credential 생성 (UI 권장)
CREATE STORAGE CREDENTIAL <name>
  WITH AWS_IAM_ROLE 'arn:aws:iam::<acct>:role/<role>'
  COMMENT 'Workshop external';

-- External Location 등록
CREATE EXTERNAL LOCATION <loc-name>
  URL 's3://<bucket>/<path>'
  WITH (STORAGE CREDENTIAL <name>);
```
