## 💻 샘플 코드

```sql
-- Genie Space에 underlying 데이터 접근 권한 부여
GRANT USE CATALOG ON CATALOG workshop_<name>   TO `analyst_group`;
GRANT USE SCHEMA  ON SCHEMA  workshop_<name>.default TO `analyst_group`;
GRANT SELECT      ON SCHEMA  workshop_<name>.default TO `analyst_group`;

-- Genie Space 자체 권한은 UI: Genie Space → Share → Add
```
