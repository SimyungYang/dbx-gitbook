## 💻 샘플 코드

```text
Budget Policies:
  - 월별 한도 / 알림
  - Cluster Policy 연동
  - Custom Tags로 추적

비용 분석 쿼리:
  SELECT usage_date, sku_name, SUM(usage_quantity * pricing.default) AS cost
  FROM system.billing.usage u JOIN system.billing.list_prices pricing USING (sku_name)
  GROUP BY usage_date, sku_name ORDER BY usage_date DESC;
```
