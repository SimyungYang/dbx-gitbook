## 💻 샘플 코드

```text
Databricks One URL: https://<workspace>/one

Persona:
  - Builder   : 모든 워크스페이스 기능 (개발자/엔지니어)
  - Consumer  : Genie + Dashboards + Apps 만 (비즈니스 사용자)

Consumer 권한만 부여하려면 Workspace Admin이:
  Workspace Settings → Users → 사용자 선택 → Persona = Consumer
```
