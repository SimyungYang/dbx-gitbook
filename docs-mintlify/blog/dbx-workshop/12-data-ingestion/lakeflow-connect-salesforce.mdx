## 💻 샘플 코드

```python
# Auto Loader 패턴
df = (
  spark.readStream
    .format('cloudFiles')
    .option('cloudFiles.format', 'json')
    .option('cloudFiles.schemaLocation', '/Volumes/workshop/_chk/schema')
    .load('/Volumes/workshop/raw/')
)

(df.writeStream
   .option('checkpointLocation', '/Volumes/workshop/_chk/state')
   .trigger(availableNow=True)
   .toTable('workshop_<name>.bronze.events'))
```
