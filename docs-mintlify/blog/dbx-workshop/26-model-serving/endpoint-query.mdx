## 💻 샘플 코드

```python
# 등록된 UC Model을 1클릭으로 서빙
# Catalog Explorer → Models → 버전 → "Serve this model"

# 또는 SDK로:
from databricks.sdk import WorkspaceClient
w = WorkspaceClient()
w.serving_endpoints.create(
    name='workshop-iris',
    config={
        'served_entities': [{
            'name': 'iris-clf',
            'entity_name': 'workshop_<name>.default.iris_clf',
            'entity_version': '1',
            'workload_size': 'Small',
            'scale_to_zero_enabled': True,
        }]
    }
)
```
