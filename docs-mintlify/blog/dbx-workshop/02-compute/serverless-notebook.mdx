## 🧭 이 기능은 무엇이고 언제 쓰나

Serverless Compute는 Databricks가 관리하는 컴퓨트로, 클러스터를 직접 만들 필요 없이 노트북에 즉시 attach 가능합니다.

# 노트북 우상단 "Connect" → "Serverless" 선택

## 🚧 자주 만나는 오류

| 에러 | 원인 | 해결 |
|------|------|------|
| `Serverless not available` | Region에서 Serverless 미지원 또는 미활성 | Workspace Admin에게 활성화 요청 |
| `Cluster start timeout` | Serverless 콜드 스타트 (드물게 30초+) | 재시도 |
