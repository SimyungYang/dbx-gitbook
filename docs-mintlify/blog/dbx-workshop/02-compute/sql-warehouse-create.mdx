## 🧭 이 기능은 무엇이고 언제 쓰나

SQL Warehouse는 BI / 분석가용 전용 컴퓨트입니다. Serverless / Pro / Classic 3종류가 있고 Photon이 항상 활성화.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
SQL → SQL Warehouses → Create SQL warehouse
  Name:        workshop-warehouse
  Cluster size: 2X-Small (DBU/h: 4)
  Type:        Serverless (권장) / Pro / Classic
  Auto stop:   10 min
  Scaling:     min 1, max 4
```

## 🚧 자주 만나는 오류

|------|------|------|
| `Quota exceeded` | Serverless quota 초과 | 더 작은 size 또는 Account Admin 문의 |
