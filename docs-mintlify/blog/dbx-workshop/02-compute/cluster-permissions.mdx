## 🧭 이 기능은 무엇이고 언제 쓰나

Cluster 권한 (Can Attach To / Can Restart / Can Manage)을 사용자/그룹에 부여합니다.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Cluster 권한 종류:
  - Can Attach To: 노트북을 cluster에 attach (실행) 가능
  - Can Restart:   cluster 재시작 / 종료 가능
  - Can Manage:    설정 변경, 권한 부여 가능 (Admin)
```
