## 🧭 이 기능은 무엇이고 언제 쓰나

SQL Warehouse 3종 비교: Serverless (Databricks 관리, 가장 빠른 시작), Pro (고객 VPC, Photon 기본), Classic (legacy, Photon 옵션).

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
| 항목           | Serverless           | Pro              | Classic         |
|----------------|----------------------|------------------|-----------------|
| 위치           | Databricks VPC       | 고객 VPC         | 고객 VPC        |
| 시작 시간      | 수초                 | 2-5분            | 4-8분           |
| Photon         | 항상                 | 항상             | 옵션            |
| Predictive IO  | 항상                 | 옵션             | X               |
| 가격           | 가장 비쌈            | 중간             | 가장 쌈         |
| 권장 용도      | BI 대시보드, Genie   | 정기 ETL/SQL     | Legacy 호환     |
```
