## 🧭 이 기능은 무엇이고 언제 쓰나

Classic All-Purpose Cluster는 고객 VPC 내 EC2에서 실행되는 전통적인 클러스터입니다. Spark 디버깅, GPU 워크로드, 특수 라이브러리 필요 시 사용.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Compute → Create compute → All-purpose compute
  Name:                 workshop-cluster
  Policy:               Unrestricted
  Access mode:          Single user (또는 Shared)
  Databricks runtime:   15.4 LTS (Photon)
  Node type:            i3.xlarge
  Autoscale:            min 2, max 4
  Auto termination:     60 minutes
```

## 🚧 자주 만나는 오류

| 에러 | 원인 | 해결 |
|------|------|------|
| `Cluster failed: INSTANCE_POOL_EXHAUSTED` | EC2 quota 초과 | 다른 instance type 시도 또는 AWS quota 증액 |
| `DBR version not available` | Region에서 미지원 | 다른 LTS 버전 선택 |
