## 🧭 이 기능은 무엇이고 언제 쓰나

Auto-scaling이 실제로 작동하는 것을 관찰합니다. 부하를 주고 worker 수 변화를 모니터링.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```python
# 부하 생성 — 큰 데이터 처리
df = spark.range(0, 10_000_000_000).repartition(200)
df.groupBy((df.id % 1000).alias("bucket")).count().count()

# 실행 중 Compute → Cluster → Metrics 탭에서 worker 수 확인
# 부하 종료 후 5분 이내 worker가 다시 줄어듦 (downscale)
```
