## 💻 샘플 코드

```text
주요 옵션:
  Databricks Runtime  — LTS 권장 (15.4 LTS / 14.3 LTS)
  Photon              — 분석 워크로드 2-3x 빠름 (ETL/SQL에 강력 권장)
  Access Mode
    - Single user      : 한 사용자 전용, UC 모든 기능, Python/Scala
    - Shared           : 다중 사용자, UC 지원, Python/SQL만
    - No isolation     : Legacy (UC 미지원)
  Autoscaling          — 부하에 따라 worker 자동 조정
  Auto termination     — Idle 시 자동 종료 (비용 절감 필수)
  Cluster tags         — 비용 추적용 (env, team, project)
```
