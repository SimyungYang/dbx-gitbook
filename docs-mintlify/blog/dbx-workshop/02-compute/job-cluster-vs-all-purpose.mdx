## 🧭 이 기능은 무엇이고 언제 쓰나

Job Cluster는 Job 실행 시 자동 생성/삭제되는 임시 클러스터로, All-Purpose보다 DBU 단가가 절반입니다. 운영 Job은 99% Job Cluster 사용.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
| 항목         | All-Purpose            | Job Cluster                 |
|--------------|------------------------|-----------------------------|
| 수명         | 사용자가 종료할 때까지 | Job 시작~종료까지           |
| DBU 단가     | 비쌈                   | 절반                        |
| 다중 사용자  | 가능 (Shared mode)     | 불가 (Job 전용)             |
| 디버깅       | 좋음 (인터랙티브)      | 한정 (Run 종료 후 확인)     |
| 비용 최적화  | Auto termination       | 자동 종료 (Job 단위)        |
| 권장 용도    | 개발/탐색              | 운영 / 스케줄 Job           |
```
