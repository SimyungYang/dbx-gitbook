## 🎯 Workshop 목적

Databricks의 모든 주요 기능을 **직접 클릭하고 코드를 실행하면서** 체감하기 위한 핸즈온 가이드입니다.
AWS Workshop 스타일로, 기능 하나하나가 단일 Lab으로 분해되어 있습니다.
