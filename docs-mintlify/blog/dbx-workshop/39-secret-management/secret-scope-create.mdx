## 💻 샘플 코드

```bash
# Secret Scope 생성
databricks secrets create-scope workshop_secrets

# Secret 등록
databricks secrets put-secret workshop_secrets api_key --string-value '<value>'

# 노트북에서 사용
# api_key = dbutils.secrets.get('workshop_secrets', 'api_key')
```
