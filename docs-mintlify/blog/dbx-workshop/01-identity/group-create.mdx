## 🧭 이 기능은 무엇이고 언제 쓰나

Account-level Group을 만들어 권한을 그룹 단위로 관리합니다. Workshop 동안 사용할 `workshop-users` Group을 생성합니다.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Account Console → User management → Groups → Add group
  Name: workshop-users
  Members:
    - workshop-test@<your-domain>.com (Lab 1.1에서 만든 User)
    - 본인 (Owner)
```

**Step 3.** Members 탭에서 User 추가

![Account-level Group 생성 — Step 3](/images/dbx-workshop/01-identity/1-2/step3-add-members.png)

## ✅ Validation

- [ ] Group이 생성됨
- [ ] Group 멤버에 본인과 테스트 User가 보임

## 🚧 자주 만나는 오류

| 에러 | 원인 | 해결 |
|------|------|------|
| `Group name already exists` | 동명 Group 존재 | 다른 이름 사용 (예: workshop-users-2026) |

## 🧹 Cleanup

Account Console → Groups → workshop-users → Delete
