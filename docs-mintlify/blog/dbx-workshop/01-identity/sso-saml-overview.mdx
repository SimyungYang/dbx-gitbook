## 🧭 이 기능은 무엇이고 언제 쓰나

SAML 2.0 기반 SSO는 Account Admin이 Account Console에서 설정합니다. 본 Lab은 UI 위치만 확인 (실제 설정은 Appendix B.6).

## 💻 샘플 코드

```text
Account Console → Settings → Security and compliance → Single sign-on (SAML)
  - SSO URL
  - Identity Provider Entity ID
  - x.509 Certificate
```
