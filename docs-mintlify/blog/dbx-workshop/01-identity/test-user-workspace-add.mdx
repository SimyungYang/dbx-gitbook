## 🧭 이 기능은 무엇이고 언제 쓰나

Identity Federation이 비활성화된 경우, Lab 1.1에서 만든 User를 Workspace에 수동 할당합니다. 활성화 상태면 자동 할당되므로 이 단계는 skip 가능.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Account Console → Workspaces → <your-workspace> → Permissions → Add user
  User: workshop-test@<your-domain>.com
  Role: User (또는 Workspace Admin)
```
