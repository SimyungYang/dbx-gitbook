## 🧭 이 기능은 무엇이고 언제 쓰나

SCIM (System for Cross-domain Identity Management)은 IdP (Okta/Azure Entra)에서 User/Group을 자동 프로비저닝합니다. 본 Lab은 UI만 확인.

## ⚙️ 핵심 옵션 / 파라미터

| 옵션 | 설명 |
|------|------|
| 본 Lab에서 다루는 핵심 변수 | 샘플 코드의 `<...>` 부분을 본인 환경에 맞게 교체 |

## 💻 샘플 코드

```text
Account Console → Settings → User Provisioning (SCIM)
  - SCIM endpoint URL
  - SCIM token (Bearer)
  - Mapping rules (User attribute → Databricks)
```
