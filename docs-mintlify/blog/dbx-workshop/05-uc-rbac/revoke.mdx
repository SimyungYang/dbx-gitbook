## 💻 샘플 코드

```sql
-- 권한 부여
GRANT <PRIVILEGE> ON <SECURABLE> <OBJECT> TO `<principal>`;

-- 권한 조회
SHOW GRANTS ON <SECURABLE> <OBJECT>;

-- 권한 회수
REVOKE <PRIVILEGE> ON <SECURABLE> <OBJECT> FROM `<principal>`;
```
