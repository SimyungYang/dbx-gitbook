## 💻 샘플 코드

```sql
-- SQL Editor에서 실행
SELECT
  o_orderstatus,
  COUNT(*) AS orders,
  SUM(o_totalprice) AS revenue
FROM samples.tpch.orders
WHERE o_orderdate >= DATE '1995-01-01'
GROUP BY o_orderstatus
ORDER BY revenue DESC;
```
