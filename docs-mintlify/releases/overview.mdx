## 릴리스 상태 정의

| 상태 | 설명 |
|------|------|
| **GA** (General Availability) | 프로덕션 사용 가능. SLA 적용. |
| **Public Preview** | 모든 고객 사용 가능하나 SLA 미적용. 프로덕션 사용 시 주의. |
| **Beta** | 초기 단계. 기능 변경 가능성 높음. 테스트 목적 권장. |
| **Private Preview** | 신청 고객만 사용 가능. SA를 통해 신청. |
