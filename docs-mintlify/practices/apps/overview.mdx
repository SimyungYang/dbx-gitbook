> Databricks Apps의 기본 개념과 아키텍처는 **[Databricks 소개 → Databricks Apps](/intro/dev-tools/databricks-apps)** 를 참고하세요.

| 가이드 | 내용 |

| [app.yaml 설정](/practices/apps/app-yaml) | 실행 커맨드, 환경변수, 리소스 |
| [인증](/practices/apps/authentication) | 앱 인증 vs 사용자 인증, 토큰 처리 |
| [SP 권한 관리](/practices/apps/service-principals) | UC 그룹, 자동화 스크립트 패턴 |
| [리소스](/practices/apps/resources) | Warehouse, Serving Endpoint, Secret 연결 |
| [배포](/practices/apps/deployment) | CI/CD, 롤링 업데이트, 롤백 |
| [업데이트](/practices/apps/updating) | 코드 업데이트, 설정 변경, 버전 관리 |
