> AI Dev Kit의 기본 개념과 설치 방법은 **[Databricks 소개 → AI Dev Kit](/intro/dev-tools/ai-dev-kit)** 를 참고하세요.

 AI Dev Kit을 실전에서 활용하기 위한 **시작 가이드, 도구 목록, Builder App 배포**를 다룹니다.

| 가이드 | 내용 |
|--------|------|
| [시작하기](/practices/ai-dev-kit/getting-started) | 첫 프로젝트 설정과 기본 워크플로우 |
| [도구 목록](/practices/ai-dev-kit/tools) | 30+ MCP 도구 카테고리별 설명 |
| [Builder App](/practices/ai-dev-kit/builder-app) | 대화형으로 앱/파이프라인 생성 |
| [배포 가이드](/practices/ai-dev-kit/deployment-guide) | deploy.sh와 CI/CD 연동 |
| [활용 사례](/practices/ai-dev-kit/use-cases) | 실전 시나리오별 워크플로우 |
