## 개요

Gold 테이블을 기반으로:
1. **AI/BI 대시보드**: 공기청정기·에어컨 운영 현황을 한눈에 보는 KPI 대시보드
2. **Genie Space**: ES 엔지니어가 자연어로 IoT 데이터를 탐색하는 공간
3. **Genie 정확도 고도화**: 가전 도메인 용어와 비즈니스 규칙으로 Genie SQL 정확도 향상

## Part A: AI/BI 대시보드

### 대시보드 1: 공기청정기 운영 현황

```
lge_appliance.gold 테이블을 사용해서 
'PuriCare 공기청정기 운영 대시보드'를 만들어줘.

포함할 위젯:
1. KPI 카운터: 전체 디바이스 수, 활성 디바이스 수, 평균 PM2.5, 필터 교체 필요 대수
2. 지역별 평균 PM2.5 지도/바차트 
3. 시간대별 공기질 추이 (PM2.5, PM10 라인 차트)
4. 운전 모드 분포 (파이 차트)
5. 필터 수명 분포 히스토그램
6. ThinQ 원격 제어 성공률 추이
7. 에너지 소비 트렌드 (일별)
```

### 대시보드 2: 에어컨 운영 현황

```
lge_appliance.gold 테이블을 사용해서 
'DUALCOOL 에어컨 운영 대시보드'를 만들어줘.

포함할 위젯:
1. KPI 카운터: 전체 디바이스 수, 평균 EER, 월간 전력 소비(kWh), 에러 발생 건수
2. 계절별 운전 모드 분포 (냉방/난방/제습)
3. 실내외 온도차 vs 전력 소비 산점도
4. 에너지 효율 등급별 소비 전력 비교
5. 컴프레서 가동률 추이
6. 에러 코드 Top 10 바차트
7. 정비 비용 월별 추이 및 보증 커버 비율
```

### 대시보드 3: 통합 가전 IoT 모니터링

```
공기청정기와 에어컨 데이터를 통합한 
'가전 IoT 통합 모니터링 대시보드'를 만들어줘.

포함할 위젯:
1. 제품군별 디바이스 건강 점수 비교
2. ThinQ 앱 사용률 및 원격 제어 통계
3. 에너지 소비 비교 (공기청정기 vs 에어컨)
4. 알림/에러 트렌드 통합 뷰
5. 지역별 디바이스 분포
```

## Part B: Genie Space

### Genie Space 1: 공기질 분석

```
'공기질 분석 Genie Space'를 만들어줘.

사용 테이블:
- lge_appliance.gold.daily_air_quality_summary
- lge_appliance.silver.ap_air_quality_clean
- lge_appliance.bronze.ap_devices

Instructions (비즈니스 규칙):
- PM2.5 기준: 0~15 좋음, 16~35 보통, 36~75 나쁨, 76~ 매우나쁨 (환경부 기준)
- PM10 기준: 0~30 좋음, 31~80 보통, 81~150 나쁨, 151~ 매우나쁨
- AQI 1~5는 각각 좋음/보통/민감군영향/나쁨/매우나쁨

샘플 질문:
- "이번 달 서울 지역 평균 PM2.5는?"
- "PM2.5가 '나쁨' 이상인 날이 가장 많은 지역은?"
- "실내 VOC가 높은 시간대는 언제야?"
```

### Genie Space 2: 에너지 효율 분석

```
'에너지 효율 분석 Genie Space'를 만들어줘.

사용 테이블:
- lge_appliance.gold.energy_comparison
- lge_appliance.silver.ac_energy_clean
- lge_appliance.silver.ap_energy_clean
- lge_appliance.bronze.ac_devices
- lge_appliance.bronze.ap_devices

Instructions:
- 에너지 효율 등급: 1등급이 가장 효율적
- EER(에너지효율비): 높을수록 효율적
- 전력 비용 계산: 1kWh = 120원 (한전 주택용 기준)

샘플 질문:
- "에어컨 모델별 평균 전력 소비 비교해줘"
- "냉방 모드에서 가장 효율적인 모델은?"
- "공기청정기 vs 에어컨 월별 전기 요금 비교"
```

### Genie Space 3: 서비스/정비 분석

```
'서비스 분석 Genie Space'를 만들어줘.

사용 테이블:
- lge_appliance.gold.alert_trend_analysis
- lge_appliance.gold.maintenance_cost_analysis
- lge_appliance.gold.filter_lifecycle_analysis
- lge_appliance.gold.device_health_score

Instructions:
- 건강 점수 80 이상: 양호, 60~79: 주의, 60 미만: 점검 필요
- 에어컨 에러코드: CH01(통신이상), CH05(실외기센서), CH10(컴프레서과부하)

샘플 질문:
- "건강 점수가 낮은 디바이스 Top 20은?"
- "에어컨 에러 코드 CH10 발생 추이는?"
- "공기청정기 HEPA 필터 평균 수명은 몇 시간이야?"
```

## Part C: Genie 정확도 고도화

```
위 3개 Genie Space에 다음을 추가해서 정확도를 높여줘:

1. 테이블 간 조인 관계 명시
   - ap_devices.device_id ↔ ap_air_quality_logs.device_id
   - ac_devices.device_id ↔ ac_temperature_logs.device_id
   
2. 자주 쓰는 쿼리 힌트 추가
   - "효율적인" → EER 값이 높은 순으로 정렬
   - "에너지 절약" → energy_saving 모드 필터
   
3. 도메인 용어 사전
   - PuriCare = 공기청정기 브랜드
   - DUALCOOL = 에어컨 브랜드
   - HEPA = High Efficiency Particulate Air (미세먼지 필터)
   - EER = Energy Efficiency Ratio
```
