---

## 개요

2026년 4월 17일, Anthropic은 **Claude Design** 을 Anthropic Labs 실험 제품으로 발표했습니다. Claude Opus 4.7 기반으로, 자연어 대화만으로 시각적 자산을 생성하고, 대화/인라인 코멘트로 수정하며, 다양한 형식으로 내보낼 수 있습니다.

---

## 핵심 특징

| 특징 | 설명 |
|------|------|
| **기반 모델** | Claude Opus 4.7 (최신 비전 모델) |
| **이용 대상** | Claude Pro, Max, Team, Enterprise 구독자 (Research Preview) |
| **생성 가능 자산** | 프로토타입, 슬라이드/피치 덱, 원페이저, 목업, 인포그래픽 |
| **수정 방식** | 자연어 대화, 인라인 코멘트, 직접 편집, 커스텀 슬라이더 |
| **디자인 시스템** | 팀의 디자인 시스템을 제공하면 자동으로 브랜드 일관성 적용 |

---

## 워크플로우

```
[자연어 설명] → [Claude가 초안 생성] → [대화/코멘트로 수정] → [내보내기 또는 Claude Code로 구현]
```

### 1단계: 설명

사용자가 원하는 디자인을 자연어로 설명합니다.

```
"SaaS 스타트업의 피치 덱을 만들어줘. 
 슬라이드는 10장, 모던한 다크 테마, 
 매출 성장 차트와 팀 소개 포함"
```

### 2단계: 초안 생성

Claude가 설명을 바탕으로 완전한 초안을 생성합니다. 레이아웃, 색상, 타이포그래피, 차트 등이 포함됩니다.

### 3단계: 반복 수정

- **자연어**: "3번째 슬라이드 차트를 바 차트에서 라인 차트로 변경해줘"
- **인라인 코멘트**: 특정 요소를 선택하고 수정 지시
- **직접 편집**: 텍스트, 색상, 위치 등을 직접 수정
- **커스텀 슬라이더**: Claude가 디자인 옵션(예: "미니멀 ←→ 풍부", "포멀 ←→ 캐주얼")을 조절할 수 있는 슬라이더를 직접 생성

### 4단계: 내보내기

| 형식 | 설명 |
|------|------|
| **내부 URL** | 팀 내 공유용 |
| **Canva** | Canva 프로젝트로 내보내기 |
| **PDF** | 문서 형식으로 저장 |
| **PPTX** | 파워포인트 형식으로 저장 |
| **HTML** | 독립 실행 가능한 HTML 파일 |
| **Claude Code** | 디자인을 Claude Code에 전달하여 실제 코드로 구현 |

## Claude Design → Claude Code 연계

Claude Design의 가장 혁신적인 기능은 **디자인에서 구현까지의 원스톱 워크플로우** 입니다.

```
[Claude Design에서 UI 프로토타입 생성]
    ↓ "이 디자인을 Claude Code로 구현해줘"
[Claude Code가 React/HTML/CSS 코드 자동 생성]
    ↓ 
[동작하는 웹 앱]
```

기존 워크플로우:
```
[디자이너: Figma에서 디자인] → [핸드오프 미팅] → [개발자: 코드로 변환] → [수정 요청] → [반복]
```

Claude Design 워크플로우:
```
[사용자: Claude Design에서 디자인] → [Claude Code로 구현] → [완료]
```

## 적합한 사용 사례

| 사용 사례 | 왜 적합한가 |
|-----------|-----------|
| **빠른 프로토타이핑** | 아이디어를 즉시 시각화하여 팀 내 공유 |
| **피치 덱** | 투자자/경영진 발표 자료를 빠르게 제작 |
| **내부 보고서** | 데이터 시각화가 포함된 원페이저 |
| **UI 목업** | 개발 전 디자인 검토용 프로토타입 |
| **소셜 미디어 콘텐츠** | 인포그래픽, 배너 등 마케팅 자산 |

### 부적합한 사용 사례

| 사용 사례 | 왜 부적합한가 | 대안 |
|-----------|-------------|------|
| **픽셀 퍼펙트 디자인** | 정밀한 픽셀 단위 제어 부족 | Figma, Sketch |
| **브랜드 가이드라인 생성** | 시스템 적용은 가능하나 생성은 제한적 | 전문 디자이너 |
| **인쇄물** | CMYK 색상, 재단선 등 인쇄 전문 기능 부재 | InDesign |

| 시나리오 | 활용 방법 |
