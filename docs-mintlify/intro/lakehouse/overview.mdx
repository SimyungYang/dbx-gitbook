---
title: "레이크하우스 아키텍처"
---


Databricks를 이해하려면 **레이크하우스(Lakehouse)** 패러다임을 먼저 알아야 합니다. 기존에는 데이터 레이크(저렴한 저장)와 데이터 웨어하우스(빠른 분석)를 별도로 운영해야 했지만, 레이크하우스는 **두 장점을 하나로 결합**한 아키텍처입니다. Databricks의 모든 기능은 이 아키텍처 위에서 동작합니다.

---

## 학습 목표
- 레이크하우스가 등장한 이유와 기존 방식의 한계 이해
- Delta Lake의 핵심 기능 (ACID 트랜잭션, 타임 트래블, 스키마 관리) 학습
- Medallion 아키텍처 (Bronze → Silver → Gold) 설계 패턴 이해
- Delta Lake와 Apache Iceberg의 관계 파악
