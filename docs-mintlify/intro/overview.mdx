**Databricks**는 데이터 엔지니어링, SQL 분석, 머신러닝, AI 에이전트를 **하나의 플랫폼**에서 수행할 수 있는 통합 데이터 인텔리전스 플랫폼입니다.

이 가이드는 Databricks를 처음 접하는 분부터 프로덕션 운영을 준비하는 분까지, **한국어로 체계적으로 학습**할 수 있도록 구성되어 있습니다.
