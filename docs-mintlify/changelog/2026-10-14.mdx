## Smart TV 워크샵 스크린샷 30장 추가

[**Section 4: Smart TV E2E 핸즈온**](/hands-on/workshop/section-4-smart-tv-handson/04-dashboard-genie) 전체에 실제 실행 결과 스크린샷이 추가되었습니다. 각 단계의 기대 결과를 시각적으로 확인하면서 실습을 진행할 수 있습니다.

- **대시보드 생성 결과 6장** — Operations, Ad Performance, Content & Engagement 대시보드 3종의 생성 완료 화면과 위젯 레이아웃을 확인할 수 있습니다
- **Genie Space 생성·설정·질의 14장** — 시청 분석, 광고 성과, 디바이스 운영 3개 Space의 생성 결과와 실제 질의 응답 화면이 추가되었습니다
- **Genie Code 고도화 Chain 6장** — 테이블 분석부터 COMMENT 적용, 샘플 질문 등록, Instructions 설정, 정확도 검증까지 Genie Code가 순차 실행하는 전체 과정을 볼 수 있습니다
- **에이전트 생성·테스트 4장** — [에이전트 개발](/hands-on/workshop/section-4-smart-tv-handson/05-agent-development) 가이드에 Knowledge Assistant 생성/테스트, Supervisor Agent 설정/결과 스크린샷이 추가되었습니다

## Genie Space UI 설정 가이드 신설

[**대시보드 & Genie Space**](/hands-on/workshop/section-4-smart-tv-handson/04-dashboard-genie) 가이드에 Genie Space를 UI에서 직접 설정하는 방법이 새로 추가되었습니다. Genie Code 없이도 기본 정보 입력, 테이블 선택, General Instructions 작성, 샘플 질문 등록까지 단계별로 따라할 수 있습니다.

## 개선 사항

- **App 배포 시 서비스 프린시펄 권한 안내** — [대시보드 & Genie Space](/hands-on/workshop/section-4-smart-tv-handson/04-dashboard-genie) 가이드의 App 배포 섹션에 서비스 프린시펄에 카탈로그 사용 권한(GRANT)을 부여해야 한다는 안내와 [권한 설정 참조 링크](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit)가 추가되었습니다
- **데이터 생성 참조 스크립트 제공** — Smart TV 실습 데이터를 생성하는 SQL·Python 스크립트(Silver/Gold 테이블, SDP 파이프라인, Knowledge Base, Supervisor Agent)가 참조 자료로 포함되었습니다. 실습 환경을 처음부터 구성할 때 활용할 수 있습니다
