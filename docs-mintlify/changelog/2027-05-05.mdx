## 업데이트

- **워크샵 AI Dev Kit 설치: Builder App `--enable-mcp` 방식으로 전환** — [AI Builder App 배포 핸즈온](/workshop/section-2-ai-builder-app/overview)과 [AI Dev Kit 설치 가이드](/workshop/section-3-ai-tools-setup/ai-dev-kit-setup)가 공식 `databricks-solutions/ai-dev-kit`의 Builder App을 `--enable-mcp` 옵션으로 배포하는 방식으로 재작성되었습니다. 한 앱에서 **Builder 웹 UI**(브라우저에서 Vibe Coding)와 **MCP 서버**(Genie Code에서 75+ 도구 사용)를 동시에 제공해, 이전에 두 개 앱을 따로 배포하던 흐름이 단일 앱·한 줄 명령으로 단순해집니다
- **외부 레포 의존 제거, Databricks 공식 리소스로 링크 일원화** — [워크샵 개요](/workshop/overview)와 관련 가이드의 참고 링크가 개인 레포 대신 [Genie Code + MCP 연결](https://docs.databricks.com/aws/en/genie-code/mcp), [Custom MCP Server 호스팅](https://docs.databricks.com/aws/en/generative-ai/mcp/custom-mcp), [AI Dev Kit GitHub](https://github.com/databricks-solutions/ai-dev-kit) 등 공식 문서·레포로 교체되었습니다. 워크샵 이후에도 최신 레퍼런스를 바로 따라갈 수 있어 장기적인 유지보수 부담이 줄어듭니다
