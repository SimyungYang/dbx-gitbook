## 교육자료 심화 시리즈 스포트라이트

**교육자료** 탭의 200페이지 이상 커리큘럼 중, 실무에 바로 적용할 수 있는 심화 시리즈를 소개합니다.

### Lakeflow Jobs 워크플로우 가이드

[**Lakeflow Jobs**](/training/05-data-engineering/lakeflow-jobs/overview) 가이드에서 데이터 파이프라인 오케스트레이션을 체계적으로 학습할 수 있습니다.

- **Job 구성** — 멀티 태스크 의존성 그래프 설계, 재시도/알림 설정, 클러스터 파라미터 최적화
- **태스크 유형 & 조건 분기** — Notebook, SQL, Python 등 태스크 유형별 활용법과 조건부 태스크 실행
- **스케줄링 & 모니터링** — 트리거 기반 스케줄링, 실행 이력 모니터링, 비용 추적

### Lakebase 가이드

[**Lakebase**](/training/11-lakebase/what-is-lakebase) 가이드에서 Databricks의 관리형 PostgreSQL 호환 OLTP 데이터베이스를 다룹니다. OLTP와 OLAP를 하나의 플랫폼에서 통합하여 데이터 복사 없이 운영 데이터를 분석에 활용하는 방법을 안내합니다.

- [설정](/training/11-lakebase/lakebase-setup) · [데이터 동기화](/training/11-lakebase/data-sync) · [Apps 연동](/training/11-lakebase/lakebase-with-apps/overview) · [프레임워크 예제](/training/11-lakebase/lakebase-with-apps/framework-examples) · [배포](/training/11-lakebase/lakebase-with-apps/deployment)

### SDP (Spark Declarative Pipelines) 가이드

[**SDP**](/training/05-data-engineering/spark-declarative-pipelines/overview) 가이드에서 선언적 데이터 파이프라인의 설계부터 운영까지 전 과정을 학습할 수 있습니다.

- **Streaming Tables & Materialized Views** — 증분 처리 패턴과 뷰 최적화
- **Expectations** — 데이터 품질 규칙 선언 및 위반 처리 전략
- **CDC & SCD** — Change Data Capture 구현과 Slowly Changing Dimensions 패턴
- [핸즈온 실습](/training/05-data-engineering/spark-declarative-pipelines/sdp-hands-on) 포함

## 모범 사례 가이드

**교육자료** 탭에 엔터프라이즈 운영을 위한 3가지 모범 사례 가이드가 포함되어 있습니다.

- [**비용 최적화**](/training/14-best-practices/cost-optimization/overview) — DBU 비용 구조, 컴퓨트 최적화, 스토리지/웨어하우스 비용 절감, 모니터링 체크리스트
- [**성능 최적화**](/training/14-best-practices/performance-tuning/overview) — Spark 쿼리 튜닝, Delta Lake 최적화, SQL Warehouse 파이프라인 튜닝, ML 진단
- [**엔터프라이즈 거버넌스**](/training/14-best-practices/enterprise-governance/overview) — 플랫폼 설계, 접근 제어, 데이터 분류, 데이터 품질 CI/CD
