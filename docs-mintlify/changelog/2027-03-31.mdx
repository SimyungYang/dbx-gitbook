## 업데이트

- **AI Dev Kit 영문 README 제공** — [AI Dev Kit](/blog/guides/ai-dev-kit/overview) 저장소에 영문 README가 기본으로 추가되었습니다. 기존 한국어 문서는 `docs/README_ko.md`로 이동했으며, 두 버전이 상단에서 상호 링크되어 있어 팀의 언어 환경에 맞게 선택해 참고할 수 있습니다
