## 공식 블로그 번역 5편 추가

[**공식 블로그 번역**](/translations/overview) 라이브러리에 최신 포스트 5편이 추가되어 총 **50편**이 되었습니다. Databricks의 최신 제품 발표와 고객 사례를 한국어로 바로 확인할 수 있습니다.

- [**Unity Catalog Business Semantics GA**](/translations/2026-04-02-unity-catalog-business-semantics-ga) — 메트릭·디멘션·규칙을 데이터 레이어에서 한 번 정의하면 모든 대시보드, SQL 쿼리, AI 에이전트가 동일한 정의를 사용합니다. Apache Spark 오픈소스 공개도 포함됩니다
- [**Lakeflow Connect Free Tier**](/translations/2026-04-02-lakeflow-connect-free-tier) — 모든 워크스페이스에서 매일 100 DBU를 무료로 제공받아, 30개 이상 SaaS 커넥터를 통해 하루 최대 1억 건의 레코드를 수집할 수 있습니다
- [**에이전틱 애널리틱스와 거버넌스**](/translations/2026-04-02-agentic-analytics-governance) — AI 에이전트가 데이터를 자율적으로 조회하는 시대에 통합된 시맨틱 레이어와 거버넌스가 왜 필수인지 다룹니다
- [**AiChemy: 신약 개발 에이전트**](/translations/2026-04-03-aichemy-drug-discovery) — MCP, Skills, 맞춤형 데이터를 활용하여 약물 후보 물질 발견을 가속하는 차세대 에이전트 사례
- [**멀티 에이전트 오디언스 인텔리전스**](/translations/2026-04-06-multi-agent-audience-intelligence) — Agent Bricks Supervisor로 자연어 캠페인 브리프를 정밀 데이터 세그먼트로 변환하는 광고 분석 사례

## 워크샵 학습 경로 개선

[**Genie Code Vibe 워크샵**](/hands-on/workshop/overview)의 학습 흐름이 더 명확해졌습니다. 처음 참여하는 분도 단계별로 따라갈 수 있습니다.

- **3단계 학습 구조** — 워크샵이 사전학습 → 환경 구성 → 핸즈온 실습 순서로 재구성되어, 각 단계의 목표와 소요 시간을 사전에 파악할 수 있습니다
- **섹션 간 자연스러운 연결** — [AI Builder App 배포](/hands-on/workshop/section-2-ai-builder-app/overview) 완료 후 [AI Dev Kit 구성](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit)으로 바로 이동하는 링크가 추가되어, 실습 흐름이 끊기지 않습니다
