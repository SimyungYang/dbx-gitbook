## 사이드바 탐색 구조 대폭 개선

사이트 전체의 탐색 구조를 개선하여 원하는 콘텐츠를 더 빠르게 찾을 수 있습니다.

### GenAI 핵심 개념 서브그룹 분리

기존에 하나의 긴 목록으로 나열되던 **Core Concepts** 섹션이 주제별 서브그룹으로 재구성되었습니다. 100페이지 이상의 콘텐츠가 다음과 같이 정리됩니다:

- **NLP에서 LLM까지** — 자연어 처리의 진화 과정
- **LLM 기초** — Transformer 구조, 학습, 추론
- **AI Agent 아키텍처** — ReAct, Tool Use, Memory, 멀티 에이전트
- **Prompt Engineering** — 기초·고급·한국어·보안
- **평가 (Evaluation)** — 메트릭, LLM-as-Judge, 파이프라인
- **Agent 프레임워크** — LangChain, CrewAI, OpenAI, Databricks AF
- **A2A 프로토콜** — Agent-to-Agent 통신
- **Agent 생태계** — 주요 기업별 에이전트 전략
- **Agent UI** — Streamlit, Gradio, Chainlit, Databricks Apps
- **AI 성숙도** — 조직 AI 활용 수준 진단

마찬가지로, [**RAG**](/blog/guides/rag/overview), [**MCP**](/blog/guides/mcp/overview), [**ML 핵심 개념**](/blog/guides/ml-fundamentals/overview) 섹션도 독립 그룹으로 분리되어 사이드바에서 바로 접근할 수 있습니다.

### 워크샵 섹션 넘버링

[**워크샵 탭**](/workshop/overview)의 섹션명에 번호가 추가되어 학습 순서를 한눈에 파악할 수 있습니다 (예: "Section 1" → "01. Genie Code 완전 가이드").

## 개선 사항

- **블로그 번역 정렬** — 공식 블로그 번역 목록이 최신순으로 재정렬되어 가장 최근 번역을 먼저 볼 수 있습니다
- **블로그 개요 페이지 수정** — 개요 페이지 제목에서 렌더링되지 않던 이미지 마크다운이 제거되었습니다
