## 업데이트

- **"시작하기" 탭을 "실전 가이드"로 재편** — 상단 탭 이름이 [실전 가이드](/blog/overview)로 바뀌었습니다. Databricks 기능 구현에 필요한 Platform Setup, AI/BI, GenAI 도구, Apps, Data Engineering 가이드만 남겨 둬 특정 기능을 구현할 때 필요한 문서를 더 빠르게 찾을 수 있습니다. 체계적인 학습 커리큘럼은 기존 [교육 자료](/training/overview) 탭에서 계속 제공됩니다
- **"AI/ML 개념" 탭 신설 — 개념 학습 콘텐츠 분리** — GenAI 핵심 개념(NLP→LLM, LLM 기초, Agent 아키텍처, Prompt Engineering, 평가, Agent 프레임워크, A2A 프로토콜, Agent 생태계, Agent UI, AI 성숙도), [RAG](/blog/guides/rag/overview), [MCP](/blog/guides/mcp/overview), [ML 핵심 개념](/blog/guides/ml-fundamentals/overview)이 새 탭 [AI/ML 개념](/blog/guides/genai-concepts/overview)으로 분리되었습니다. 개념 학습과 실전 가이드를 오가기 쉬워지고, 이전보다 탭별 분량이 균형 있게 정리되었습니다
- **AI 동향 탭 그룹 세분화** — 기존 단일 그룹이 [개요](/ai-trends/overview), [2026년 4월 핫 키워드](/ai-trends/hot-keywords-2026-04), [기업별 동향 분석](/ai-trends/anthropic-trends-2026) 3개 그룹으로 나뉘었습니다. 시장 개요, 최신 키워드, 기업별 심층 분석을 목적에 맞게 빠르게 찾아갈 수 있습니다
