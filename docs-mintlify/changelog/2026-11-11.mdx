## 교육자료: AI 에이전트 개발 심화 (7편)

[**AI 에이전트**](/training/10-agent-development/what-is-ai-agent) 교육 섹션에 에이전트 개발의 전 과정을 다루는 7편의 심화 가이드가 공개되었습니다. AI 에이전트가 무엇인지부터 프로덕션 배포까지, 단계별로 학습할 수 있습니다.

- [**AI 에이전트란?**](/training/10-agent-development/what-is-ai-agent) — 챗봇과 에이전트의 차이, 에이전트의 핵심 구성 요소(LLM 두뇌 + 도구 + 메모리)를 설명합니다
- [**Vector Search & RAG 파이프라인**](/training/10-agent-development/vector-search) — 의미 기반 유사도 검색과 검색 증강 생성(RAG) 파이프라인의 오프라인/온라인 구조를 다룹니다
- [**에이전트 구축 & 도구 연결**](/training/10-agent-development/building-agents) — Mosaic AI Agent Framework와 ChatAgent 인터페이스로 에이전트를 구현하고, [Unity Catalog 함수를 도구로 활용](/training/10-agent-development/tools-and-functions)하여 거버넌스를 유지하는 방법을 안내합니다
- [**에이전트 평가**](/training/10-agent-development/agent-evaluation) — LLM-as-Judge 방식의 의미적 평가와 Mosaic AI Agent Evaluation을 활용한 품질 측정 체계를 설명합니다
- [**Agent Bricks**](/training/10-agent-development/agent-bricks) — 코드 없이 문서 Q&A, SQL 분석, 멀티 에이전트 오케스트레이션을 빠르게 구축하는 사전 구축 에이전트 템플릿을 소개합니다

## 교육자료: 데이터 웨어하우징 심화 (7편)

[**데이터 웨어하우징**](/training/06-data-warehousing/databricks-sql-overview) 교육 섹션에 Databricks SQL을 활용한 분석 환경 구축을 위한 7편의 가이드가 추가되었습니다.

- [**Databricks SQL 개요**](/training/06-data-warehousing/databricks-sql-overview) — 레이크하우스 위에서 웨어하우스급 SQL 성능을 제공하는 DBSQL의 아키텍처와 핵심 개념을 설명합니다
- [**SQL 주요 기능 & 윈도우 함수**](/training/06-data-warehousing/sql-language-features) — 파이프 구문, QUALIFY, [윈도우 함수](/training/06-data-warehousing/window-functions)(순위·집계·분포), [SQL 스크립팅](/training/06-data-warehousing/sql-scripting)(변수·조건문·반복문)까지 고급 SQL 기능을 다룹니다
- [**테이블과 뷰**](/training/06-data-warehousing/tables-and-views) — Managed/External 테이블과 일반 뷰의 차이, 그리고 [구체화된 뷰(Materialized View)](/training/06-data-warehousing/materialized-views)로 쿼리 결과를 사전 계산하여 성능을 극대화하는 방법을 안내합니다
- [**쿼리 최적화**](/training/06-data-warehousing/query-optimization) — Query Profile 분석, 조인 전략, 파티션 프루닝 등 실행 시간과 DBU 비용을 동시에 줄이는 실전 기법을 제공합니다

## 교육자료: 보안과 거버넌스 심화 (9편)

[**보안과 거버넌스**](/training/12-security-and-governance/security-overview) 교육 섹션에 엔터프라이즈 보안 운영을 위한 9편의 가이드가 공개되었습니다.

- [**보안 개요 & CMK 암호화**](/training/12-security-and-governance/security-overview) — 공유 책임 모델(Shared Responsibility Model)과 [고객 관리 키(CMK)](/training/12-security-and-governance/cmk-encryption)를 사용한 데이터 암호화 전략을 설명합니다
- [**감사 로그 & 네트워크 보안**](/training/12-security-and-governance/audit-logs) — [감사 테이블 구조와 실전 쿼리](/training/12-security-and-governance/audit-logs), VNet Injection·Private Link·IP Access List 등 [네트워크 수준 방어선](/training/12-security-and-governance/network-security) 구성을 다룹니다
- [**System Tables**](/training/12-security-and-governance/system-tables/overview) — 4편의 가이드로 주요 시스템 테이블 구조, 비용·사용량 분석 쿼리, 모니터링 대시보드 구축 방법을 안내합니다
- [**워크스페이스 관리**](/training/12-security-and-governance/workspace-admin) — 사용자 권한 체계, 환경 분리 전략, 비용 통제 방안을 포함한 운영 가이드를 제공합니다

[**데이터 엔지니어링**](/training/05-data-engineering/data-engineering-overview) 교육 섹션에 파이프라인 설계와 스트리밍 처리를 위한 핵심 가이드가 추가되었습니다.
