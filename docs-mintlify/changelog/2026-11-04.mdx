## 교육자료 커리큘럼 전면 공개

[**교육자료**](/training/02-databricks-overview/what-is-databricks) 탭에 Databricks 플랫폼 전 영역을 다루는 **212페이지 커리큘럼**이 완성되었습니다. 처음 접하는 분부터 운영 담당자까지, 역할과 수준에 맞춰 필요한 내용을 찾아 학습할 수 있습니다.

### 플랫폼 기초 (8편)

[**Databricks 플랫폼**](/training/02-databricks-overview/what-is-databricks) 섹션에서 Databricks가 무엇이고, 왜 필요하며, 어떻게 시작하는지를 다룹니다.

- **경쟁 환경 & 가격 구조** — Snowflake, BigQuery, Fabric과의 비교와 DBU 기반 과금 체계를 설명합니다
- **아키텍처 & UI 투어** — Control Plane/Data Plane 구조와 Workspace UI 주요 메뉴를 안내합니다
- **노트북 기초·고급** — 셀 실행부터 위젯, `%run`, `dbutils`, Git 연동까지 노트북 활용 전 과정을 다룹니다

[**Unity Catalog**](/training/07-unity-catalog/what-is-unity-catalog) 섹션에서 3-Level 네임스페이스부터 접근 제어, 데이터 리니지, 데이터 공유까지 통합 거버넌스의 전체 그림을 다룹니다.

- **데이터 리니지 & 공유** — [리니지 추적과 Impact Analysis](/training/07-unity-catalog/data-lineage/overview), [Delta Sharing](/training/07-unity-catalog/delta-sharing), [Marketplace](/training/07-unity-catalog/marketplace), [Clean Rooms](/training/07-unity-catalog/clean-rooms)를 활용한 조직 간 데이터 협업
