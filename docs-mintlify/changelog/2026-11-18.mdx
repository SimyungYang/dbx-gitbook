## 교육자료: Auto Loader 심화 (4편)

[**Auto Loader**](/intro/data-engineering/auto-loader/overview) 교육 섹션에 클라우드 스토리지 파일 수집의 핵심 이론을 다루는 4편의 심화 가이드가 공개되었습니다.

- [**Auto Loader란?**](/intro/data-engineering/auto-loader/what-is-auto-loader) — 클라우드 스토리지에 새로 도착한 파일을 자동으로 감지·수집하는 증분 파일 수집 기능의 핵심 개념을 설명합니다
- [**파일 감지 모드 비교**](/intro/data-engineering/auto-loader/file-notification-vs-directory-listing) — Directory Listing과 File Notification 두 가지 감지 모드의 차이점, 적합한 시나리오, 성능 특성을 비교합니다
- [**스키마 추론과 진화**](/intro/data-engineering/auto-loader/schema-inference) — 스키마 자동 추론, 새 컬럼 추가 시 자동 진화, 스키마 힌트 설정 방법을 안내합니다
- [**주요 옵션 가이드**](/intro/data-engineering/auto-loader/auto-loader-options) — `cloudFiles.format`, 파일 필터링, 최대 파일 수 제한 등 실전에서 자주 사용하는 설정 옵션을 정리합니다

## 교육자료: Spark Declarative Pipelines (7편)

[**Spark Declarative Pipelines (SDP)**](/intro/data-engineering/spark-declarative-pipelines/overview) 교육 섹션에 선언적 파이프라인 프레임워크의 전체 과정을 다루는 7편의 가이드가 추가되었습니다.

- [**SDP란?**](/intro/data-engineering/spark-declarative-pipelines/what-is-sdp) — 이전 Delta Live Tables(DLT)에서 이름이 변경된 선언적 파이프라인의 핵심 개념과 명령형 파이프라인과의 차이를 설명합니다
- [**Streaming Tables & Materialized Views**](/intro/data-engineering/spark-declarative-pipelines/streaming-tables-and-mvs) — SDP에서 사용하는 두 가지 데이터셋 유형의 특성과 사용 시나리오를 비교합니다
- [**Expectations**](/intro/data-engineering/spark-declarative-pipelines/expectations) — 선언적 데이터 품질 규칙으로 잘못된 데이터를 자동으로 감지·처리하는 방법을 안내합니다
- [**파이프라인 구성**](/intro/data-engineering/spark-declarative-pipelines/pipeline-configuration) — 클러스터 설정, 라이브러리 의존성, 개발/프로덕션 모드 전환 등 파이프라인 운영에 필요한 구성 옵션을 다룹니다
- [**Event Log 활용**](/intro/data-engineering/spark-declarative-pipelines/event-log) — 파이프라인 실행 이력, 데이터 품질 메트릭, 성능 진단에 활용하는 이벤트 로그 분석 방법을 제공합니다
- [**CDC와 SCD**](/intro/data-engineering/spark-declarative-pipelines/cdc-and-scd) — `APPLY CHANGES`를 사용한 Change Data Capture 처리와 SCD Type 1/2 구현 방법을 설명합니다
- [**SDP 핸즈온**](/intro/data-engineering/spark-declarative-pipelines/sdp-hands-on) — 메달리온 아키텍처 기반 파이프라인을 직접 구축하는 실습 가이드를 제공합니다

## 교육자료: Lakeflow Connect (3편)

[**Lakeflow Connect**](/intro/data-engineering/lakeflow-connect/overview) 교육 섹션에 외부 데이터 소스 수집 서비스를 다루는 3편의 가이드가 추가되었습니다.

- [**Lakeflow Connect란?**](/intro/data-engineering/lakeflow-connect/what-is-lakeflow-connect) — MySQL, PostgreSQL, Salesforce 등 운영 DB와 SaaS 애플리케이션에서 데이터를 코드 없이 수집하는 관리형 서비스의 개념을 설명합니다
- [**수집 파이프라인 설정**](/intro/data-engineering/lakeflow-connect/ingestion-pipeline-setup) — 소스 연결, 대상 테이블 매핑, 스케줄 구성까지 수집 파이프라인을 단계별로 설정하는 방법을 안내합니다
- [**Lakeflow Connect 핸즈온**](/intro/data-engineering/lakeflow-connect/lakeflow-connect-hands-on) — 실제 데이터 소스에서 레이크하우스까지 증분 수집 파이프라인을 구축하는 실습 가이드를 제공합니다

## 교육자료: Lakeflow Jobs (10편)

[**Lakeflow Jobs**](/intro/data-engineering/lakeflow-jobs/overview) 교육 섹션에 워크플로 오케스트레이션의 전체 과정을 다루는 10편의 심화 가이드가 공개되었습니다.

- [**Lakeflow Jobs란?**](/intro/data-engineering/lakeflow-jobs/what-is-lakeflow-jobs) — DAG 기반 오케스트레이션 서비스의 핵심 개념과 아키텍처를 설명합니다
- **Job 설정 4편** — [Task 유형과 의존성](/intro/data-engineering/lakeflow-jobs/job-configuration/tasks-and-dependencies), [재시도와 알림](/intro/data-engineering/lakeflow-jobs/job-configuration/retry-and-alerts), [클러스터와 파라미터](/intro/data-engineering/lakeflow-jobs/job-configuration/cluster-and-parameters), [핸즈온과 최적화](/intro/data-engineering/lakeflow-jobs/job-configuration/hands-on-and-optimization)까지 Job 구성 전 과정을 다룹니다
- [**Task 유형 상세**](/intro/data-engineering/lakeflow-jobs/task-types) — Notebook, Python, SQL, SDP, dbt 등 지원되는 Task 유형별 특성과 사용 시나리오를 비교합니다
- [**조건부 Task**](/intro/data-engineering/lakeflow-jobs/conditional-tasks) — IF/ELSE 분기, 동적 Task 생성 등 파이프라인 흐름을 유연하게 제어하는 방법을 안내합니다
- [**스케줄링과 트리거**](/intro/data-engineering/lakeflow-jobs/scheduling-and-triggers) — Cron 스케줄, 파일 도착 트리거, 연속 실행 모드 등 다양한 실행 방식을 설명합니다
- [**모니터링과 알림**](/intro/data-engineering/lakeflow-jobs/monitoring-and-alerting) — 실행 이력 분석, 실패 알림 설정, System Tables를 활용한 비용·성능 추적 방법을 제공합니다

## 개선 사항

- **Lakehouse Monitoring** — [머신러닝 교육자료](/intro/ml/lakehouse-monitoring)에 Lakehouse Monitoring 가이드가 추가되어, Delta 테이블의 데이터 품질 프로파일링, 분포 드리프트 감지, ML 모델 성능 추적을 자동화하는 방법을 확인할 수 있습니다
- **개발 도구: AI Dev Kit & Genie Code** — [개발 도구 부록](/intro/dev-tools/ai-dev-kit)에 [AI Dev Kit](/intro/dev-tools/ai-dev-kit)과 [Genie Code](/intro/dev-tools/genie-code) 교육자료가 추가되어, MCP 기반 AI 코딩 어시스턴트 연동과 노트북 내장 AI 어시스턴트의 활용법을 학습할 수 있습니다
