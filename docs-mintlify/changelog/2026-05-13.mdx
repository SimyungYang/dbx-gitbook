이번 주는 신규 기능 추가 없이 기존 콘텐츠의 안정화에 집중한 기간입니다. 지난 4주간 500페이지 이상의 대규모 콘텐츠가 출시되었으며, 이번 주에는 별도의 제품 변경 사항이 없습니다.

최근 추가된 주요 콘텐츠를 아직 확인하지 못하셨다면:

- [**RAG 종합 가이드**](/blog/guides/rag/overview) — 임베딩, 벡터 검색, 하이브리드 검색, 한국어 NLP까지 22페이지 심화 가이드
- [**GenAI 핵심 개념**](/blog/guides/genai-concepts/overview) — NLP 진화, LLM 기초, 에이전트 아키텍처, 프롬프트 엔지니어링 등 10개 주제 40페이지 이상
- [**교육자료**](/training/02-databricks-overview/what-is-databricks) — Databricks 플랫폼 전체를 다루는 200페이지 이상의 한국어 커리큘럼
- [**Smart TV 워크샵**](/blog/hands-on/smart-tv-vibe/overview) — 3개 학습 트랙의 E2E 핸즈온 실습
