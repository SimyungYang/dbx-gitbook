## Genie Code 심화 콘텐츠 추가

[**Genie Code 완전 가이드**](/workshop/section-1-vibe-coding-intro/genie-code/what-is-genie-code)에 실무에서 바로 활용할 수 있는 심화 콘텐츠가 추가되었습니다.

- **비용 구조** — AI 영역(무료)과 컴퓨트 영역(과금)의 명확한 구분. 무료 작업(코드 생성, 설명, 리팩토링)과 과금 작업(데이터 조회, Agent Mode 실행)을 한눈에 비교할 수 있습니다
- **Multi-Agent 아키텍처** — Task Planner → Model Router → Tool Executor 4레이어 구조와 내장 도구 ~50개의 동작 방식을 설명합니다
- **LLM 모델 상세** — Azure OpenAI GPT + Anthropic Claude 자동 선택 방식, Zero Data Retention 정책, TLS/AES-256 암호화 등 보안 사양을 정리했습니다
- **Chat Mode vs Agent Mode 비교** — 동작 방식, 사용자 개입 수준, 적합한 작업 유형, 컨텍스트 범위를 비교하여 모드 선택을 돕습니다

## 워크샵 탐색 구조 개선

[**핸즈온(SmartTV) 탭**](/workshop/overview)의 탐색 구조가 개선되어 학습 콘텐츠를 더 쉽게 찾을 수 있습니다.

- **Genie Code 가이드 서브그룹 분리** — 11개 페이지가 "프롬프트 가이드"와 "주의사항 & 참고" 두 그룹으로 재구성되었습니다
- **페이지 번호 추가** — 각 페이지 제목에 순서 번호가 추가되어(01~10) 학습 순서를 바로 파악할 수 있습니다
- **섹션명 통일** — 모든 섹션이 "Section 1/2/3/4" 형식으로 정리되었습니다
