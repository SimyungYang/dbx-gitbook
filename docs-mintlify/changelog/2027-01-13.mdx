## 개선 사항

- **AI Gateway 아키텍처 다이어그램 개선** — [AI Gateway 가이드](/hands-on/workshop/section-3-ai-tools-setup/ai-gateway) 페이지의 아키텍처 다이어그램이 ASCII 박스에서 Mermaid 플로우차트로 교체되었습니다. 사용자 → AI Gateway → Foundation Models 흐름이 시각적으로 더 명확하게 표현됩니다
- **AI Gateway 가이드 범용화** — 동일 페이지에서 특정 조직명 참조가 제거되고 범용 시나리오로 변경되었습니다. "도입 시나리오 예시"로 제목이 수정되어, 어떤 조직에서든 바로 참고할 수 있습니다
- **워크샵 실습 코드 별도 저장소 공개** — [Smart TV 핸즈온](/hands-on/workshop/section-4-smart-tv-handson/01-setup) 실습 코드와 [AI Dev Kit](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 배포 스크립트(`deploy.sh`)가 각각 독립 저장소로 분리·공개되었습니다. 워크샵 참가자가 코드를 바로 클론하여 실습을 시작할 수 있습니다
