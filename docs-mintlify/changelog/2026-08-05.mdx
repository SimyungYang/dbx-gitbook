## 핸즈온: Smart TV Vibe Coding 워크샵 — 모듈별 상세 안내

[**AI Vibe Coding Workshop**](/blog/hands-on/smart-tv-vibe/overview)의 8개 핸즈온 모듈을 소개합니다. 170만 건의 가상 스마트TV 데이터를 기반으로, Databricks 핵심 기능 12가지를 하루 만에 체험하는 풀데이 워크샵입니다.

- [**환경 설정 & 데이터 기반**](/blog/hands-on/smart-tv-vibe/00-setup) — AI Dev Kit 설치, Unity Catalog 네임스페이스 구성, 170만 건 합성 데이터 자동 생성, Delta Lake Time Travel·Schema Evolution 체험
- [**데이터 엔지니어링**](/blog/hands-on/smart-tv-vibe/02-data-engineering) — Bronze-Silver-Gold 메달리온 파이프라인 구축, SDP(Lakeflow Declarative Pipelines) 기반 증분 처리, Expectations를 통한 데이터 품질 자동 검증
- [**분석 & 대시보드**](/blog/hands-on/smart-tv-vibe/03-analytics) — AI/BI Dashboard 4페이지(10개 이상 차트), Genie Space 생성 및 17가지 비즈니스 질문으로 자연어 데이터 탐색
- [**실시간 스트리밍**](/blog/hands-on/smart-tv-vibe/04-streaming) — Databricks App 기반 이벤트 생성기, UC Volume 랜딩존, Auto Loader 실시간 수집, SDP 스트리밍 파이프라인 구성
- [**ML 개인화 추천**](/blog/hands-on/smart-tv-vibe/05-ml) — ALS 협업 필터링, MLflow 실험 추적, Unity Catalog Model Registry 등록, Vector Search 콘텐츠 임베딩, Model Serving 엔드포인트 배포
- [**GenAI & 에이전트**](/blog/hands-on/smart-tv-vibe/06-genai) — Vector Search + Foundation Model API 기반 RAG, Knowledge Assistant·Genie Agent·Supervisor Agent 멀티 에이전트 오케스트레이션, MLflow Tracing 및 Agent Evaluation
- [**Apps & Lakebase 배포**](/blog/hands-on/smart-tv-vibe/07-apps-lakebase) — Gradio 챗봇·Streamlit 대시보드·FastAPI 포털 3종 앱 배포, Lakebase(PostgreSQL OLTP)로 사용자 선호도·추천 캐시·피드백 저장

3개 학습 트랙(A: 노트북 실행, B: [Claude Code + AI Dev Kit MCP](/blog/hands-on/smart-tv-vibe/notebooks/track-b-ai-guided/PROMPT_GUIDE), C: [Genie Code Agent Mode](/blog/hands-on/smart-tv-vibe/notebooks/track-c-genie-code/GENIE_CODE_GUIDE))으로 참가자 수준에 맞게 진행할 수 있습니다.

## 가이드: GenAI 기초 시리즈 — NLP에서 LLM까지

[**GenAI 핵심 개념**](/blog/guides/genai-concepts/overview) 섹션의 기초 시리즈 2종을 소개합니다. LLM의 역사적 맥락과 핵심 원리를 체계적으로 이해할 수 있습니다.

### NLP에서 LLM까지

[**NLP 발전사**](/blog/guides/genai-concepts/nlp-evolution/overview) 가이드에서 규칙 기반 시스템(1950~)부터 Transformer(2017)까지, 각 기술이 왜 등장하고 어떤 한계가 다음 혁신을 이끌었는지를 6개 페이지에 걸쳐 다룹니다.

- [**임베딩 혁명**](/blog/guides/genai-concepts/nlp-evolution/embeddings) — Word2Vec, GloVe로 단어를 벡터 공간에 매핑하여 의미 연산이 가능해진 전환점
- [**Attention & Transformer**](/blog/guides/genai-concepts/nlp-evolution/attention-transformer) — Self-Attention 메커니즘이 RNN/LSTM의 순차 처리 한계를 해결한 과정
- [**사전학습 시대**](/blog/guides/genai-concepts/nlp-evolution/pretrain-era) — BERT, GPT, T5로 이어지는 대규모 사전학습 패러다임의 등장

### LLM 기초

[**LLM 기초 가이드**](/blog/guides/genai-concepts/llm-basics/overview) 8개 페이지에서 "다음 토큰 예측"이라는 단순한 원리가 어떻게 범용 AI 능력으로 발현되는지를 다룹니다.

- [**Transformer 아키텍처**](/blog/guides/genai-concepts/llm-basics/transformer) — Self-Attention, Multi-Head Attention, Positional Encoding의 핵심 구조
- [**학습 과정**](/blog/guides/genai-concepts/llm-basics/training) — Pre-training → SFT → RLHF/DPO로 이어지는 3단계 학습 파이프라인
- [**환각 현상**](/blog/guides/genai-concepts/llm-basics/hallucination) — Hallucination의 유형·원인·해결 방법(RAG, Grounding) 및 산업별 리스크
- [**추론 모델**](/blog/guides/genai-concepts/llm-basics/reasoning) — o1, o3, R1 등 Test-time Compute Scaling 기반의 추론 특화 모델 동향
