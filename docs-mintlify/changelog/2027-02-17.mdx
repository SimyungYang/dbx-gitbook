## 새 기능

- **AI Dev Kit 간편 배포 (deploy.sh)** — [Genie Code + AI Dev Kit 구성 가이드](/workshop/section-3-ai-tools-setup/genie-code-aidevkit)에서 안내하는 AI Dev Kit MCP 서버를 이제 `deploy.sh` 스크립트 한 줄로 배포할 수 있습니다. 앱 생성, 배포, 서비스 프린시펄 권한 부여, Skills 설치를 자동으로 처리하며, 전제조건(CLI 버전, 인증, jq 등)도 상세히 안내합니다

## 업데이트

- **Smart TV 워크샵 3단계 구조 개편** — [Smart TV 핸즈온](/workshop/section-4-smart-tv-handson/01-setup) 워크샵이 **사전 학습 → 환경 구성 → 본 실습** 3단계로 재구성되었습니다. 각 단계별 소요 시간과 선행 요건이 명확히 표시되어, 참가자가 실습 전 준비 상태를 스스로 점검할 수 있습니다
- **Genie Code 가이드 보강** — [Genie Code 완전 가이드](/workshop/section-1-vibe-coding-intro/genie-code/what-is-genie-code) 페이지에 비용 구조, 아키텍처, LLM 상세 정보, Chat vs Agent Mode 비교가 추가되었습니다. Genie Code의 내부 동작 방식과 과금 체계를 이해하는 데 도움이 됩니다
- **워크샵 스크린샷 15장 추가** — Smart TV 핸즈온 전 과정에 실제 UI 스크린샷이 삽입되었습니다. 기존 텍스트 플레이스홀더를 대체하여, 각 단계에서 기대되는 화면을 바로 확인할 수 있습니다
- **워크샵 범용화 완료** — 전체 워크샵 자료에서 특정 조직·날짜 참조가 제거되어, 어떤 팀에서든 바로 활용할 수 있는 범용 교육 자료로 정리되었습니다
