## 개선 사항

- **Genie Code 접근 방법 확대** — [Genie Code 시작하기](/hands-on/workshop/section-1-vibe-coding-intro/genie-code/getting-started) 가이드에 Genie Code를 여는 3가지 방법이 추가되었습니다. 이제 상단 바 버튼, 노트북 사이드바, `Cmd+I` 인라인 호출 중 상황에 맞는 방법을 선택할 수 있으며, 노트북이 아닌 페이지에서도 Genie Code를 바로 사용할 수 있다는 안내가 포함되었습니다
- **AI Dev Kit 스크린샷 정리** — [AI Dev Kit 구성 가이드](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit)의 모든 스크린샷에서 브라우저 상단 UI(탭, 주소 표시줄, 북마크 바)가 제거되어, Databricks 화면만 깔끔하게 표시됩니다
- **워크샵 범용화 완료** — [Smart TV E2E 핸즈온](/hands-on/workshop/section-4-smart-tv-handson/01-setup)의 환경 설정 가이드에서 Genie Code 접근 설명이 "노트북 우측 상단"에서 "어느 페이지에서든 가능"으로 수정되었고, 전체 워크샵 자료에서 특정 조직 관련 잔여 참조가 모두 정리되었습니다
