## Smart TV Vibe Coding 워크샵 확장

[**Smart TV 워크샵**](/blog/hands-on/smart-tv-vibe/overview)이 **3개 학습 트랙**을 갖춘 풀데이 워크샵으로 확장되었습니다. 170만 건의 가상 스마트TV 로그 데이터를 기반으로 10개 모듈을 실습합니다.

- **Track A** — 노트북 기반 실습 (전통적 방식)
- **Track B** — Claude Code + AI Dev Kit MCP 활용 ([프롬프트 가이드](/blog/hands-on/smart-tv-vibe/notebooks/track-b-ai-guided/PROMPT_GUIDE))
- **Track C** — Genie Code Agent Mode 전용 ([Genie Code 가이드](/blog/hands-on/smart-tv-vibe/notebooks/track-c-genie-code/GENIE_CODE_GUIDE))

10개 모듈: 환경 설정 → 데이터 생성 → SDP 파이프라인 → 대시보드 & Genie → 스트리밍 → ML 추천 → GenAI 에이전트 → Apps & Lakebase 배포

전용 **워크샵 탭**에서는 [Genie Code 완전 가이드](/workshop/section-1-vibe-coding-intro/genie-code/overview) (11페이지), [AI Builder App 배포](/workshop/section-2-ai-builder-app/overview), [AI 도구 환경 구성](/workshop/section-3-ai-tools-setup/overview) (AI Gateway · Claude Code · AI Dev Kit)을 포함한 구조화된 커리큘럼을 제공합니다.
