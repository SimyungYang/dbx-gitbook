## 업데이트

- **Genie Code 접근 방법 3종 안내** — [Genie Code 시작하기](/workshop/section-1-vibe-coding-intro/genie-code/getting-started)와 [Smart TV Section 1: 환경 설정](/workshop/section-4-smart-tv-handson/01-setup)에서 Genie Code에 접근하는 방법이 **상단 바 버튼 / 노트북 사이드바 / `Cmd+I` 인라인** 세 가지로 정리되었습니다. 기존에는 "노트북 우측 상단 아이콘"만 안내하고 있어 참가자가 다른 진입 경로를 놓치곤 했는데, 이제 어느 화면에서든 본인에게 맞는 방식으로 Genie Code를 열 수 있습니다
- **Smart TV 워크샵 범용 정리** — 워크샵 콘텐츠에 남아 있던 특정 조직(LGE)·특정 트랙(Track C) 참조가 제거되었습니다. [Smart TV 핸즈온](/workshop/overview) 전체가 업종·조직에 상관없이 그대로 활용할 수 있는 범용 자료로 마무리되어, 사내 교육이나 고객 워크샵에 별도 수정 없이 바로 투입할 수 있습니다
