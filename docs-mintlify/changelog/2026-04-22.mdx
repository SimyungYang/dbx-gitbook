## 교육 커리큘럼 전면 공개

**교육자료** 탭에 Databricks 플랫폼 전체를 아우르는 200페이지 이상의 한국어 교육 커리큘럼이 공개되었습니다. 초급부터 고급까지, 실무 중심으로 구성되어 있습니다.

- **데이터 기초 & 선행 지식** — RDB, 빅데이터 생태계, 배치/스트리밍 개념부터 차근차근 시작
- **Databricks 플랫폼 개요** — 아키텍처, 경쟁력 비교, 가격 체계, 노트북 기초/고급
- **레이크하우스 아키텍처** — Delta Lake, Liquid Clustering, UniForm, Deletion Vectors, Predictive Optimization
- **컴퓨트** — 클러스터 유형/정책, Instance Pools, Photon, Serverless, SQL Warehouse
- **데이터 엔지니어링** — Auto Loader, Structured Streaming, SDP(구 DLT), Lakeflow Connect/Jobs
- **데이터 웨어하우징** — SQL 기능, Window Functions, Pivot/Unpivot, Materialized Views, AI Functions
- **Unity Catalog** — 접근 제어, Row Filters, Column Masks, Data Lineage, Delta Sharing, Volumes
- **AI/BI** — Lakeview Dashboards, Genie, Metric Views, 하이브리드 BI 전략
- **머신러닝** — AutoML, MLflow(Tracking·Registry·Tracing·Evaluation), Feature Engineering, Model Serving, AI Gateway
- **AI 에이전트** — Vector Search, RAG 파이프라인, Agent 빌딩/평가/배포, Agent Bricks
- **Lakebase** — 설정, 데이터 동기화, Apps 연동
- **보안 & 거버넌스** — SSO, SCIM, CMK, 감사 로그, System Tables, 네트워크 보안
- **모범 사례** — 비용 최적화, 성능 튜닝, 엔터프라이즈 거버넌스
