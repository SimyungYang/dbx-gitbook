## 가이드: Genie Code + AI Dev Kit — A-to-Z 통합 가이드 완전 재작성

[**Genie Code에 AI Dev Kit 구성하기**](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드가 전면 재작성되었습니다. 사전 지식 없이 처음부터 끝까지 따라할 수 있는 자립형 가이드로 재구성했습니다.

- **전체 워크플로우 포함** — git clone → CLI 인증 → 앱 생성 → 배포 → 권한 설정 → Skills 구성 → MCP 연결 → 도구 테스트까지 모든 단계의 명령어를 포함
- **크로스 프로덕트 역할 명확화** — Genie Code 단독(단일 제품 영역)과 AI Dev Kit 연동(여러 제품에 걸친 작업) 시의 차이를 비교 테이블로 정리. Genie Space 생성, 대시보드·Job 스케줄링, Knowledge Assistant·Supervisor Agent 구성 등 44개 도구를 한 대화에서 호출할 수 있습니다
- **깨진 다이어그램 수정** — ASCII box diagram을 테이블 형식으로 교체하여 모든 환경에서 정상 렌더링
