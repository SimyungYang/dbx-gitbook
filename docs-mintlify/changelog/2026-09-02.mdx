## 워크샵 타임테이블 분리

[**Genie Code Vibe 워크샵**](/hands-on/workshop/overview)에 풀데이·반나절 두 가지 타임테이블이 별도 페이지로 제공됩니다. 조직의 일정에 맞는 포맷을 선택하여 워크샵을 운영할 수 있습니다.

- [**풀데이 (8시간)**](/hands-on/workshop/timetable-fullday) — Section 1~3 환경 구성 + 핸즈온 전체(데이터 생성, SDP 파이프라인, 대시보드 & Genie Space, GenAI 에이전트 개발)
- [**반나절 (4시간)**](/hands-on/workshop/timetable-halfday) — 핵심 요약 + 핸즈온 집중. AI Builder App 배포(Section 2) 생략, 에이전트 개발은 Supervisor 통합까지 진행
