## MLOps 핸즈온 독립 탭 신설

[**예지보전 MLOps 핸즈온**](/blog/hands-on/predictive-maintenance/overview)이 기존 "시작하기" 탭에서 분리되어 독립 탭으로 승격되었습니다. 사이드바에서 바로 접근할 수 있어 MLOps 실습 콘텐츠를 더 빠르게 찾을 수 있습니다.

- Feature Engineering부터 모델 모니터링, Job 스케줄링까지 **10단계 E2E 파이프라인**을 별도 탭에서 집중 학습
- 시작하기 탭의 가이드 목록이 간결해져 전체 네비게이션 탐색이 수월해졌습니다

## Genie Code Vibe 탭에 종합 가이드 통합

[**종합 가이드 (3-Track, Streaming, ML)**](/blog/hands-on/smart-tv-vibe/overview)가 Genie Code Vibe 탭 하위로 통합되었습니다. 기존 Section 4와 중복되던 페이지를 제거하고, 고유 콘텐츠만 유지하여 학습 동선이 명확해졌습니다.

- **Track A: 노트북 실행** — Foundation, Streaming, ML, Apps & Lakebase 4개 핵심 모듈로 정리
- **Track B: Claude Code + AI Dev Kit** — AI 가이드 기반 프롬프트 실습 유지
- 이전에 분산되어 있던 워크샵 콘텐츠가 한 탭에서 모두 접근 가능합니다
