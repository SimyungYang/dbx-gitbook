## Genie Code 기술 심화 콘텐츠

[**Genie Code란?**](/workshop/section-1-vibe-coding-intro/genie-code/what-is-genie-code) 페이지에 비용 구조, 내부 아키텍처, Chat vs Agent Mode 비교가 추가되었습니다. 기존 4줄 요약이 실무에서 바로 활용할 수 있는 상세 가이드로 확장됩니다.

- **비용 구조 테이블** — AI 영역(무료)과 컴퓨트 영역(기존 사용료)을 구분하고, 무료 작업 6가지와 과금 작업 3가지를 표로 정리
- **Multi-Agent 아키텍처** — Task Planner → Model Router → Tool Executor 4레이어 구조와 LLM 모델(Azure OpenAI GPT + Anthropic Claude) 자동 선택, Zero Data Retention 보안 상세
- **Chat Mode vs Agent Mode 비교** — 동작 방식, 사용자 개입, 적합 작업, 컨텍스트 범위, Slash 명령 지원 여부를 한눈에 비교

## Genie Code 접근 방법 가이드 개선

[**시작하기**](/workshop/section-1-vibe-coding-intro/genie-code/getting-started) 페이지에서 Genie Code를 여는 방법이 3가지로 정리되었습니다. 기존에는 "노트북을 열어 아이콘 클릭" 한 가지만 안내했으나, 이제 상단 바 버튼·노트북 사이드바·`Cmd+I` 인라인 호출까지 모든 접근 경로를 표로 제공합니다.

## AI Dev Kit MCP 테스트 프롬프트 100선

[**Genie Code에 AI Dev Kit 구성하기**](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에 연결되는 AI Dev Kit MCP 도구별 테스트 프롬프트 100개가 카테고리별로 정리되었습니다. 데이터 탐색, 파이프라인, ML, 거버넌스 등 영역별로 바로 실행할 수 있는 프롬프트를 제공하여 MCP 연동 후 도구 동작을 빠르게 검증할 수 있습니다.

## 개선 사항

- **워크샵 초심자 친화성 강화** — 네비게이션 흐름, 전제조건, 용어 설명이 보강되어 Databricks를 처음 접하는 사용자도 워크샵을 따라갈 수 있습니다
- **MCP 트러블슈팅 가이드** — AI Dev Kit MCP 연결 시 자주 발생하는 문제(cold start, 도구 호출 실패)에 대한 대응 방법이 추가되었습니다
- **Custom Instructions 확장** — Genie Code [Custom Instructions](/workshop/section-1-vibe-coding-intro/genie-code/custom-instructions)에 MCP 도구 사용 규칙 6가지와 quarantine 스키마 네이밍 규칙이 추가되었습니다
