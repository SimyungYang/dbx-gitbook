## 네비게이션 탭 재구성

사이트 상단 탭 구조가 콘텐츠 성격에 맞게 재편되었습니다. 핸즈온 워크샵을 더 빠르게 찾을 수 있습니다.

- **MLOps 핸즈온 탭 신설** — [예지보전 MLOps](/hands-on/predictive-maintenance/overview) 핸즈온이 "시작하기" 탭에서 분리되어 별도 **MLOps 핸즈온** 탭으로 이동했습니다. MLOps 실습에 바로 접근할 수 있습니다
- **AI Vibe Workshop 통합** — [AI Vibe Workshop (3-Track)](/blog/hands-on/smart-tv-vibe/overview)이 [Genie Code Vibe](/hands-on/workshop/overview) 탭에 통합되어, Genie Code 관련 워크샵 자료를 한 곳에서 확인할 수 있습니다
