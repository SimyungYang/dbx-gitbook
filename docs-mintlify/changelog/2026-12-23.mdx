## 개선 사항

- **AI Dev Kit 가이드 간소화** — [Genie Code에 AI Dev Kit 구성하기](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드가 기존 9단계에서 5단계로 대폭 간소화되었습니다. `deploy.sh` 간편 배포를 기본 경로로 안내하고, 중복되던 수동 설정 단계(MCP 서버 연결, 도구 선택, 테스트)를 통합하여 처음 사용하는 분도 빠르게 따라갈 수 있습니다
- **핵심 정리 테이블 업데이트** — 가이드 하단의 구성 요소 요약 테이블이 `deploy.sh` 기반 워크플로우를 반영하도록 수정되었습니다. Skills 배포가 deploy.sh에 의해 자동 처리된다는 점이 명시되어, 별도 수동 배포가 불필요함을 확인할 수 있습니다
- **트러블슈팅 참조 번호 정리** — 가이드 구조 변경에 맞춰 트러블슈팅 섹션의 Step 참조 번호가 모두 업데이트되었습니다. 문제 해결 시 정확한 단계를 바로 찾을 수 있습니다
