## 개선 사항

- **AI Dev Kit 구성 가이드 간소화** — [Genie Code에 AI Dev Kit 구성하기](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드가 기존 9단계에서 5단계로 재구성되었습니다. `deploy.sh` 간편 배포를 기본 경로로 설정하여 배포 → 카탈로그 GRANT → MCP 연결 → 도구 선택 → 테스트 순서로 자연스럽게 진행됩니다. 중복되던 MCP 서버 연결, 도구 선택, 테스트 단계(기존 Step 7~9)가 제거되어 가이드가 더 간결해졌습니다
- **트러블슈팅 참조 업데이트** — 가이드 하단의 트러블슈팅 테이블에서 Step 참조 번호가 새 구조에 맞게 수정되었습니다. 오류 발생 시 올바른 단계를 빠르게 찾아갈 수 있습니다
- **핵심 정리 테이블 명확화** — MCP(AI Dev Kit) 설정 방식 안내가 "Step 1~8 수동 구성"에서 "`deploy.sh` 한 줄 또는 수동 구성"으로 변경되어, 간편 배포가 기본 옵션임을 명확히 합니다
