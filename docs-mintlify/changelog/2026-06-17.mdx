## 사이드바 내비게이션 전면 개편

모든 탭의 사이드바가 **중첩 서브그룹** 구조로 재편되어, 원하는 콘텐츠를 더 빠르게 찾을 수 있습니다.

### 시작하기 탭

- **Platform Setup** — [AWS Workspace 구성](/blog/platform-setup/aws/overview), [Azure Workspace 구성](/blog/platform-setup/azure/overview), [인증(Authentication)](/blog/guides/platform-setup/authentication/overview) 3개 서브그룹으로 분리
- **AI/BI & Analytics** — [Genie Space 운영 가이드](/blog/guides/genie-space/overview)와 [플랫폼 비교](/blog/guides/platform-comparison/overview)를 별도 서브그룹으로 구성
- **GenAI & Agent** — [Genie Code](/blog/guides/genie-code/overview), [Agent Bricks](/blog/guides/agent-bricks/overview), [AI Dev Kit](/blog/guides/ai-dev-kit/overview), [GenAI 도구 생태계](/blog/guides/genai-tools/overview) 4개 서브그룹으로 분리
- **GenAI 핵심 개념** — NLP에서 LLM까지, LLM 기초, Agent 아키텍처, Prompt Engineering 등 10개 주제별 서브그룹으로 세분화
- **RAG** — [RAG 핵심 개념](/blog/guides/rag/concepts/overview)과 [고급 검색 전략](/blog/guides/rag/advanced-retrieval/overview)을 별도 서브그룹으로 구성
- **Hands-on Workshop** — [예지보전 MLOps](/blog/hands-on/predictive-maintenance/overview)와 [AI Vibe Workshop](/blog/hands-on/smart-tv-vibe/overview)을 별도 서브그룹으로 분리

### 교육자료 탭

- **레이크하우스 아키텍처** — Delta Lake, Iceberg & UniForm을 별도 서브그룹으로 구성
- **컴퓨트** — 클러스터 설정, 클러스터 정책을 별도 서브그룹으로 분리
- **데이터 엔지니어링** — Auto Loader(실습 포함), Spark Structured Streaming, SDP, Lakeflow Connect, Lakeflow Jobs(Job 설정 포함) 6개 서브그룹으로 세분화
- **데이터 웨어하우징** — SQL 기능, Pivot & Unpivot을 서브그룹으로 구성
- **Unity Catalog** — 스토리지 & 외부 연결, 접근 제어, 데이터 리니지, 데이터 공유, Volumes를 별도 서브그룹으로 분리
- **머신러닝** — 하이퍼파라미터 튜닝, MLflow(Model Registry, MLflow Tracing 포함), Model Serving, Feature Engineering을 별도 서브그룹으로 구성
- **보안과 거버넌스** — ID & 접근 관리, System Tables를 서브그룹으로 분리
- **모범 사례** — 비용 최적화, 성능 튜닝, 엔터프라이즈 거버넌스를 별도 서브그룹으로 구성
