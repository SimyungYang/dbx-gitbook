## 신규 가이드

### Genie Code 심화 가이드

Genie Code 활용을 극대화할 수 있는 3개의 심화 가이드를 추가했습니다.

- [**AI Dev Kit 스킬 연동**](/blog/guides/genie-code/ai-dev-kit-skills) — AI Dev Kit으로 도메인 지식(메달리온 아키텍처, 파이프라인 구성 등)을 Genie Code에 주입하는 방법
- [**설정 & 모범사례**](/blog/guides/genie-code/settings-best-practices) — 역할별 최적 설정과 운영 모범사례 종합 가이드
- [**MCP 연동**](/blog/guides/genie-code/mcp) — MCP를 통해 Genie Code를 외부 도구, 데이터 소스, 워크플로우와 연결하는 방법

### 플랫폼 운영 가이드

- [**인증 종합 가이드**](/blog/guides/platform-setup/authentication/overview) — OAuth U2M/M2M, PAT Federation 등 Databricks 인증 방식 전체 안내
- [**Databricks Geo (데이터 거주)**](/blog/guides/platform-setup/databricks-geos) — 클라우드 리전 그룹핑과 데이터 거주 관리
- [**Lakeflow 증분 처리**](/blog/guides/lakeflow-incremental/overview) — Streaming Table/MV의 Full Refresh 방지 전략 및 CDC 패턴

### GenAI 도구 & MCP

- [**GenAI 도구 생태계**](/blog/guides/genai-tools/overview) — AI 코딩 어시스턴트, LLM 플랫폼, 노코드 빌더, AI 관측성 도구 종합 가이드
- [**MCP 가이드**](/blog/guides/mcp/overview) — Model Context Protocol 설정, 인기 서버, Databricks MCP, 모범사례

### Smart TV Vibe Coding 워크샵

[**Genie Code + AI Dev Kit 기반 E2E 핸즈온 워크샵**](/workshop/overview)이 추가되었습니다. Smart TV 데이터 플랫폼 시나리오를 기반으로 아래 과정을 진행합니다:

- Genie Code 완전 가이드 (프롬프트 원칙, 템플릿, 커스텀 인스트럭션, 보안 제한사항)

- AI Gateway + Claude Code + AI Dev Kit 환경 구성
- 데이터 생성 → SDP 파이프라인 → 대시보드/Genie → 에이전트 개발 전체 실습
