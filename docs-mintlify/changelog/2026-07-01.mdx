## 콘텐츠 통합 관리 체계 구축

기존에 5개 독립 저장소로 분산 관리되던 모든 콘텐츠가 **단일 저장소로 통합**되었습니다. 교육자료, 가이드, 블로그 번역, AI 동향, 워크샵 콘텐츠를 하나의 저장소에서 관리하게 되어, 콘텐츠 간 일관성이 향상되고 업데이트가 더 빠르게 반영됩니다.

- **통합 대상** — 교육자료(Training), 가이드(Blog), 블로그 번역(Translations), AI 동향(AI Trends), 워크샵(Workshop) 5개 영역
- **사용자 영향** — 콘텐츠 간 상호 참조 링크가 안정적으로 유지되며, 전체 사이트의 업데이트 주기가 단축됩니다

## 워크샵 콘텐츠 범용화 완료

[**Smart TV 핸즈온 워크샵**](/workshop/overview) 전체에서 특정 기업 참조가 완전히 제거되어, 누구나 동일한 실습 자료로 학습할 수 있게 되었습니다.

- **디바이스 명칭 통일** — 모든 TV 모델명, 디바이스 ID, 광고주 목록이 범용 Smart TV 시나리오에 맞게 정리되었습니다
- **에이전트 시나리오 업데이트** — [에이전트 개발](/workshop/section-4-smart-tv-handson/05-agent-development) 모듈의 사운드바, 스트리밍 채널 등 참조가 일반화되었습니다
- **프로젝트 메타데이터 정리** — 모든 내부 프로젝트 파일에서 특정 기업 워크샵 문구가 범용 교육 자료 문구로 교체되었습니다

## AI Dev Kit 구성 가이드 시각 자료 보강

[**Genie Code에 AI Dev Kit 구성하기**](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에 실제 설정 과정을 보여주는 **9개 스크린샷**이 추가되었습니다.

- **설정 플로우 시각화** — App 배포부터 MCP 서버 연결까지 전체 과정을 캡처한 스크린샷으로 단계별 진행이 직관적으로 확인 가능
- **이미지 품질 개선** — 브라우저 크롬(주소창, 탭 등)을 제거하여 핵심 UI 요소에 집중할 수 있도록 정리

## 개선 사항

- **Genie Code 접근 방법 수정** — [워크샵 시작 가이드](/workshop/section-1-vibe-coding-intro/genie-code/getting-started)에서 Genie Code 접근 방법이 3가지(상단 바 버튼, 노트북 사이드바, `Cmd+I` 인라인)로 정확하게 안내됩니다
- **MCP 설정 용어 정리** — [AI Dev Kit 환경 구성](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에서 "Genie Code 단독" 표현을 "MCP 없이"로 수정하여 의미를 명확히 했습니다
