## Genie Code + AI Dev Kit 통합 가이드 재작성

[**Genie Code에 AI Dev Kit 구성하기**](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드가 A-to-Z 완전 자립형 가이드로 재작성되었습니다. CLI 인증부터 앱 배포, MCP 서버 연결, 도구 테스트까지 사전 지식 없이 따라할 수 있도록 전 과정을 단계별로 안내합니다.

- **크로스 프로덕트 작업 비교표** — Genie Code 단독 vs AI Dev Kit 연동 시 가능한 작업 범위를 한눈에 비교
- **구성 아키텍처 정리** — Built-in 기능, MCP Tools(44개 도구), Workspace Skills 세 가지 확장 경로를 테이블로 정리
- **Step-by-step 배포 절차** — Databricks CLI 인증 → 앱 생성 → 환경 변수 → 배포 → MCP 서버 등록 → Skills 배포 → 연결 확인까지 전 과정 포함

## 워크샵 범용화 완성

워크샵 전체에서 특정 조직·일자 관련 참조가 제거되어 누구나 재활용할 수 있는 범용 교육 자료로 정리되었습니다.

- **타이틀 정리** — 모든 워크샵 페이지에서 번호 접두사("01.", "02." 등)가 제거되어 네비게이션이 깔끔해졌습니다
- **다이어그램 개선** — 깨지던 ASCII 다이어그램이 테이블 형식으로 교체되어 모든 환경에서 올바르게 표시됩니다
