## 개선 사항

- **AI Dev Kit 수동 설정 단계 완전 제거** — [Genie Code에 AI Dev Kit 구성하기](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에서 CLI 인증, 앱 생성, 코드 배포, SP 권한, Skills 배포 등 수동 설정 6단계가 모두 제거되었습니다. `deploy.sh`가 전체 과정을 자동화하므로 더 이상 수동 경로를 참조할 필요가 없습니다
- **도구 선택 가이드 추가** — AI Dev Kit이 제공하는 44개 도구 중 워크샵에 권장하는 15개를 정리한 테이블이 추가되었습니다. Genie Code가 기본 제공하는 기능과 중복되는 도구(`ask_genie`, `query_vs_index`, `execute_sql`)는 OFF를 권장하여, 20개 도구 제한 안에서 효율적으로 구성할 수 있습니다
- **AI Builder App 배포 시간 단축** — [AI Builder App 배포 핸즈온](/hands-on/workshop/section-2-ai-builder-app/overview) 페이지의 예상 소요 시간이 30분에서 5분으로 변경되었습니다. `deploy.sh`를 활용한 간편 배포 경로가 안내되며, 기존 수동 과정은 참고용으로만 유지됩니다
- **워크샵 타임테이블 조정** — [풀데이 워크샵](/hands-on/workshop/timetable-fullday) 소요 시간이 8시간에서 7시간으로 조정되었습니다
