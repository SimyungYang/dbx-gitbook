## 개선 사항

- **AI Dev Kit 구성 가이드 재정비** — [Genie Code에 AI Dev Kit 구성하기](/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에서 `deploy.sh` 간편 배포 이후 잘못 끼어있던 수동 설정 단계를 제거하고, Step 1(deploy.sh) → 2(GRANT) → 3(MCP 연결) → 4(도구 선택) → 5(테스트)의 깔끔한 흐름으로 재구성했습니다. 처음 따라 하는 분도 혼동 없이 진행할 수 있습니다
- **워크샵 스크린샷 안내 개선** — [Genie Code 완전 가이드](/workshop/section-1-vibe-coding-intro/genie-code/overview) 및 [AI 도구 환경 구성](/workshop/section-3-ai-tools-setup/overview) 등 15개 페이지에서 스크린샷 플레이스홀더 57건을 구체적인 화면 참고 안내로 변환했습니다. 각 단계에서 확인해야 할 UI 위치가 텍스트로 명확히 안내됩니다
- **AI Builder App 간편 배포 안내 추가** — [Section 2: AI Builder App 배포](/workshop/section-2-ai-builder-app/overview)에 `deploy.sh`를 활용한 간편 배포 참조가 추가되었습니다
- **풀데이 워크샵 시간 수정** — [워크샵 개요](/workshop/overview) 타임테이블이 8시간에서 7시간으로 정정되었습니다. 실제 진행 시간에 맞춰 안내합니다
