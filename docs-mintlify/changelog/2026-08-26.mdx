## 가이드: Genie Code — AI 코딩 어시스턴트 완벽 가이드

[**Genie Code 사용 가이드**](/practices/genie-code/overview) 7개 페이지에서 Databricks 내장 AI 코딩 어시스턴트의 활용법을 체계적으로 다룹니다. Unity Catalog 메타데이터를 자동으로 인식하고, Agent 모드에서 다단계 작업을 자율 수행하는 Genie Code의 실전 활용 가이드입니다.

- [**활용 시나리오**](/practices/genie-code/scenarios) — EDA 자동화, 대시보드 생성, 디버깅 등 ROI가 높은 사용 패턴 Top 5와 구체적인 대화 예시
- [**프롬프트 쿡북**](/practices/genie-code/prompt-cookbook) — 데이터 엔지니어링, SQL 분석, ML, GenAI 등 전 영역에서 바로 사용할 수 있는 실전 프롬프트 레시피북
- [**MCP 연동**](/practices/genie-code/mcp) — Slack, JIRA, GitHub 등 외부 도구를 MCP로 연결하여 Genie Code를 워크스페이스 경계를 넘는 AI 에이전트로 확장
- [**설정 & 모범사례**](/practices/genie-code/settings-best-practices) — Custom Instructions, Agent 모드 설정, 팀 단위 운영 전략
- [**AI Dev Kit 스킬 연동**](/practices/genie-code/ai-dev-kit-skills) — AI Dev Kit의 30개 이상 도구를 Genie Code에서 자연어로 호출
