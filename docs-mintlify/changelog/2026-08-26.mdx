## 워크샵 타임테이블 — 풀데이 & 반나절 분리

[**Genie Code Vibe 워크샵**](/workshop/overview)의 타임테이블이 별도 페이지로 분리되었습니다. 조직의 가용 시간에 맞춰 두 가지 포맷 중 선택할 수 있습니다.

- [**풀데이 (8시간)**](/workshop/timetable-fullday) — Section 1~3 환경 구성 후, 5개 핸즈온 모듈을 모두 진행하는 종일 과정
- [**반나절 (4시간)**](/workshop/timetable-halfday) — 핵심 요약 + 핸즈온 집중 과정. AI Builder App 배포(Section 2)는 생략하고, 에이전트 개발은 Supervisor 통합까지 진행

각 포맷별 시간 배분과 생략 항목이 명확히 안내되어, 워크샵 기획 시 바로 참고할 수 있습니다.

---

## 개선 사항

- **워크샵 범용화 완료** — 워크샵 전체에서 특정 조직 대상 정보가 제거되어, 누구나 활용할 수 있는 범용 교육 자료로 정비되었습니다. Smart TV와 가전 IoT 케이스 모두 제품 브랜드에 중립적인 표현으로 통일되었습니다
- **Genie Code 접근 방법 업데이트** — [워크샵 Genie Code 가이드](/workshop/section-1-vibe-coding-intro/genie-code/getting-started)에서 Genie Code 접근 방법이 3가지(상단 바 버튼, 노트북 사이드바, Cmd+I 인라인)로 구체화되었습니다. 노트북뿐 아니라 어느 페이지에서든 접근할 수 있다는 점이 명확히 안내됩니다
