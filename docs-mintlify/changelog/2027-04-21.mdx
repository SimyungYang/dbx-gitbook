## 업데이트

- **로드맵 GA 기능에 공식 문서 링크 연결** — [Unity Catalog](/releases/roadmap/unity-catalog), [Data Engineering](/releases/roadmap/data-engineering), [Databricks SQL](/releases/roadmap/databricks-sql), [AI/ML](/releases/roadmap/ai-ml), [Compute & Serverless](/releases/roadmap/compute-serverless), [Apps & Lakebase](/releases/roadmap/apps-lakebase), [Security](/releases/roadmap/security) 등 7개 로드맵 페이지의 모든 GA 기능 설명에 Databricks 공식 문서 링크가 걸렸습니다. 릴리스 현황 표에서 바로 해당 기능의 공식 레퍼런스로 이동해 세부 스펙을 확인할 수 있습니다
- **코딩 에이전트 비교 표 확장 — Cursor · Windsurf · Kiro 추가** — [Agentic Engineering](/ai-ml/trends/keywords/agentic-engineering) 페이지의 코딩 에이전트 비교에 **Cursor**(VS Code 포크 기반 AI-native IDE, Background Agent), **Windsurf**(Codeium, Flow/Cascade 자동화), **Kiro**(Amazon, Spec-Driven Development 내장)가 추가되었습니다. Claude Code의 기반 모델 표기도 Opus 4.7로 업데이트되고, Codex의 Desktop/병렬 에이전트 특성이 반영되어 최신 코딩 에이전트 생태계를 한눈에 비교할 수 있습니다
