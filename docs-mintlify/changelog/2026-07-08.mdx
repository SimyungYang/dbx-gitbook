## 교육자료: 에이전트 배포 파이프라인 심화

[**에이전트 배포**](/intro/agents/agent-deployment/overview) 가이드에서 AI 에이전트를 프로덕션에 안전하게 배포하는 전체 파이프라인을 4개 페이지에 걸쳐 다룹니다. 개발 완료된 에이전트를 실 서비스에 올리기까지 필요한 모든 단계를 순서대로 안내합니다.

- [**모델 로깅과 등록**](/intro/agents/agent-deployment/logging-and-registration) — MLflow로 에이전트를 로깅하고 Unity Catalog에 등록하는 방법. `ChatModel` 인터페이스, 시그니처 정의, 의존성 관리까지 포함
- [**엔드포인트 생성과 Review App**](/intro/agents/agent-deployment/endpoint-and-review) — Model Serving 엔드포인트 배포와 `agents.deploy()`를 통한 Review App 자동 생성. 팀 내 베타 테스트 및 피드백 수집 체계 구축
- [**버전 관리와 모니터링**](/intro/agents/agent-deployment/versioning-and-monitoring) — LLM 드리프트, 데이터 드리프트 등 AI 특유의 비결정성에 대응하는 버전 관리 전략과 Inference Table 기반 모니터링
- [**Guardrails와 트러블슈팅**](/intro/agents/agent-deployment/guardrails-and-troubleshooting) — AI Gateway를 통한 입출력 보호, PII 마스킹, 토픽 차단, 그리고 배포 시 자주 발생하는 문제의 해결 방법

## 교육자료: 보안 & ID 관리 심화 가이드

**교육자료** 탭의 [보안과 거버넌스](/intro/security/security-overview) 섹션에서 엔터프라이즈 환경에 필수적인 ID 관리와 보안 구성을 심도 있게 다루는 4개의 가이드를 소개합니다.

- [**SSO 설정**](/intro/security/sso-setup) — SAML 2.0과 OIDC의 차이점, Entra ID·Okta별 설정 절차, 트러블슈팅 체크리스트
- [**SCIM 프로비저닝**](/intro/security/scim-provisioning) — IdP에서 Databricks로 사용자·그룹을 자동 동기화하는 설정 방법. 수동 관리의 보안 위험을 제거합니다
- [**서비스 프린시펄**](/intro/security/service-principals) — CI/CD 파이프라인과 자동화 시스템을 위한 전용 계정 생성, OAuth 인증 구성, PAT 대비 보안 이점
- [**시크릿 관리**](/intro/security/secret-management) — Databricks-backed와 Key Vault-backed Scope의 차이, CLI를 통한 시크릿 관리, ACL 설정으로 접근 제어

## 교육자료: Auto Loader 핸즈온 실습

[**Auto Loader 실습**](/intro/data-engineering/auto-loader/auto-loader-hands-on/overview) 시리즈에서 클라우드 스토리지의 증분 파일 수집을 처음부터 프로덕션 수준까지 4단계로 실습합니다.

- [**실습 1: CSV 수집**](/intro/data-engineering/auto-loader/auto-loader-hands-on/setup-and-csv) — 환경 구성, 증분 수집 기본 동작, 체크포인트 이해
- [**실습 2: JSON & 스키마 진화**](/intro/data-engineering/auto-loader/auto-loader-hands-on/json-and-schema-evolution) — JSON 수집, 스키마 진화 대응, 중첩 구조 처리
- [**실습 3: SDP 통합**](/intro/data-engineering/auto-loader/auto-loader-hands-on/sdp-integration) — `read_files()` 활용, Medallion 파이프라인 구성, 데이터 품질 규칙 적용
- [**실습 4: 검증 & 트러블슈팅**](/intro/data-engineering/auto-loader/auto-loader-hands-on/validation-and-troubleshooting) — 품질 리포트 분석, 체크포인트 리셋, 일반적인 에러 해결 방법

## 개선 사항

- **교육자료 일괄 품질 개선** — 모노레포 통합 과정에서 200페이지 이상의 [교육자료](/intro/databricks/what-is-databricks) 전체가 일관된 포맷으로 정리되었습니다. 프론트매터, 내부 링크, 코드 블록 형식이 통일되어 더 안정적인 읽기 경험을 제공합니다
- **소스 콘텐츠 버전 관리 통합** — 기존에 별도 관리되던 교육자료·블로그 번역·AI 동향·워크샵의 원본 소스가 배포 콘텐츠와 동일한 저장소에서 관리됩니다. 콘텐츠 업데이트 주기가 단축되고 변경 추적이 용이해졌습니다
