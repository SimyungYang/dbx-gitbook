## AI Dev Kit MCP 통합 가이드 독립 배포

[**Genie Code + AI Dev Kit 연동 가이드**](/blog/guides/ai-dev-kit/getting-started)를 독립 저장소로 제공하여, 별도의 환경에서도 바로 활용할 수 있게 되었습니다. CLI 인증부터 MCP 서버 배포, 도구 활성화까지 전 과정을 하나의 가이드에서 완료할 수 있습니다.

- **배포 가능한 MCP 서버 앱** — `app.yaml`과 `main.py`가 포함된 프로덕션 레디 앱을 제공합니다. Databricks Apps에 바로 배포하여 Genie Code에서 AI Dev Kit 도구를 사용할 수 있습니다
- **100개 테스트 프롬프트** — Genie Space 관리, Supervisor Agent, Knowledge Assistant, Job 스케줄링, 대시보드, 파이프라인 등 카테고리별로 정리된 프롬프트로 AI Dev Kit MCP 연동을 바로 검증할 수 있습니다
- **MCP 우선 사용 지침 템플릿** — Genie Code User Instructions에 MCP 도구를 노트북 코드보다 우선 사용하도록 설정하는 권장 지침이 제공됩니다

## 역할별 MCP 도구 선택 프로필

[**AI Dev Kit 도구 가이드**](/blog/guides/ai-dev-kit/tools)에 5가지 역할별 MCP 도구 프로필이 추가되었습니다. 모든 도구를 한꺼번에 활성화하지 않고, 자신의 역할에 맞는 15개 도구만 선택하여 효율적으로 사용할 수 있습니다.

- **Data Engineer** — SDP 파이프라인, Job 스케줄링, 카탈로그 관리 중심
- **AI/ML Engineer** — Model Serving, Vector Search, Agent 개발 중심
- **Data Analyst** — Genie Space, 대시보드, SQL 쿼리 중심
- **Platform Admin** — 워크스페이스, 클러스터, 권한 관리 중심
- **All-rounder** — 전 영역을 아우르는 범용 프로필

## Smart TV 워크샵 독립 실행 지원

[**Genie Code Vibe 워크샵**](/workshop/overview)의 Smart TV E2E 핸즈온이 독립 저장소로도 제공됩니다. 워크샵 진행자가 별도로 클론하여 자체 환경에서 바로 실습을 진행할 수 있습니다.

- **범용화 완료** — 특정 조직 참조가 모두 제거되어 어떤 워크스페이스에서든 실행할 수 있습니다
- **Genie Code 접근 방법 최신화** — 상단 바 아이콘, 노트북 사이드바, `Cmd+I` 인라인 호출 3가지 방법이 반영되었습니다
