## 새 기능

- **AI Dev Kit 간편 배포** — [Genie Code에 AI Dev Kit 구성하기](/hands-on/workshop/section-3-ai-tools-setup/genie-code-aidevkit) 가이드에 `deploy.sh` 기반 원커맨드 배포가 추가되었습니다. 기존에 7단계 수동 설정이 필요했던 MCP 앱 생성, 소스 코드 배포, 서비스 프린시펄 권한 부여, Skills 배포를 한 번에 자동으로 처리합니다. 설정 소요 시간이 약 30분에서 약 5분으로 단축됩니다

## 개선 사항

- **전제조건 명확화** — AI Dev Kit 구성 가이드에 Databricks CLI 버전, OAuth 인증, Python, jq 등 필요한 사전 요구사항이 표로 정리되었습니다. 설정 전에 누락된 도구를 빠르게 확인할 수 있습니다
- **가이드 구조 개선** — 기존 수동 설정 단계(Step 1~6)가 "상세 설정" 하위 섹션으로 정리되었습니다. 간편 배포를 사용하는 경우 해당 섹션을 건너뛰고 바로 MCP 서버 연결 단계로 이동할 수 있습니다
