## 새로운 기능

- **AI Dev Kit 원클릭 배포 스크립트** — AI Dev Kit MCP 서버를 Databricks에 배포하는 `deploy.sh` 스크립트가 추가되었습니다. 앱 생성, 배포, Service Principal 권한 설정, Skills 등록까지 전체 과정이 자동화되어 약 3분 만에 완료됩니다. DAB(Databricks Asset Bundles) 기반 `databricks.yml` 설정도 함께 제공됩니다

## 개선 사항

- **AI Gateway 아키텍처 시각화 개선** — [AI Gateway 가이드](/workshop/section-3-ai-tools-setup/ai-gateway) 페이지의 아키텍처 다이어그램이 Mintlify 네이티브 Steps/Cards 컴포넌트로 교체되었습니다. 사용자(AI 도구) → AI Gateway(Rate Limit·비용 추적·감사 로그) → Foundation Models 3단계 흐름이 아이콘과 설명이 포함된 인터랙티브 카드로 표현되어, 한눈에 구조를 파악할 수 있습니다
- **Smart TV 워크샵 전면 재구성** — 워크샵 교육 자료가 사전학습 / 환경구성 / 핸즈온 실습 3단계 구조로 개편되었습니다. [Genie Code 가이드](/workshop/section-1-vibe-coding-intro/genie-code/overview)에 비용 구조, 아키텍처, LLM 상세 정보, Chat vs Agent 모드 비교가 추가되고, 15개 실제 스크린샷이 삽입되어 초심자도 쉽게 따라갈 수 있습니다
- **워크샵 범용화 완료** — Smart TV 워크샵에서 모든 특정 조직 참조가 제거되었습니다. 어떤 팀에서든 바로 활용할 수 있는 범용 교육 자료로 정리되었습니다
