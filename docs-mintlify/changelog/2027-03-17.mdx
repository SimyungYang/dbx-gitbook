- **AI 동향: 2026년 4월 핫 키워드** — [2026년 4월 AI 핫 키워드](/ai-ml/trends/hot-keywords-2026-04) 페이지가 추가되었습니다. ## 버그 수정

- **AI Dev Kit: Windows Git Bash 경로 변환 수정** — Windows Git Bash(MINGW64) 환경에서 `/Workspace/...` 경로가 `C:/Program Files/Git/Workspace/...`로 잘못 변환되어 `databricks workspace import-dir` 명령이 실패하던 문제가 수정되었습니다. macOS/Linux에서는 영향 없습니다
